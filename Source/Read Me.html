This document defines the goals for version 2.0

 

Maintainability

  • Implement Syntax Analysis
  • Implement Code Analysis

Modularity

  • Expose sub-components (like the HTML parser) to public

Performance

  • Reduce memory footprint

Conformance

  • Pass ACID1 Test

 

Implementation

HTML5 Parsing

Fully compatible parsing of HTML5 and HTML4 documents. Parsing of FRAMESET excluded as well as any SCRIPT related functionality.

HTML DOM

Partial DOM Level 2 Core

Implement partial DOM Level 2 Core specification. Implement interfaces and functionality that is related to read-only accessing the parsed document.
