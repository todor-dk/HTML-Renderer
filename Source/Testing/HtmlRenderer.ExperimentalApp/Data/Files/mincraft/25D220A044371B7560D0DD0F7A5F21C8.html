Minecraft

Minecraft is a sandbox indie game written in Java originally created by Markus "Notch" Persson, and now by his company Mojang AB. It is heavily inspired by Infiniminer. Minecraft is a game about breaking and placing blocks. At first, people built structures to protect against nocturnal monsters, but as the game grew players worked together to create wonderful, imaginative things. It allows the players to explore, interact with, and modify a dynamically-generated map made of one-cubic-meter-sized blocks. In addition to blocks, the environment features plants, mobs, and items. Some activities in the game include mining for ore, fighting hostile mobs, and crafting new blocks and tools by gathering various resources found in the game.
