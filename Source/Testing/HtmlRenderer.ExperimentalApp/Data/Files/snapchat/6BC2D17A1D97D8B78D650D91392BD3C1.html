Get Snapchat for the Kindle Fire, HD, and HDX

Posted on January 22, 2014 by snapchatkindlefire.com

Learn how to install Snapchat onto your first generation Kindle Fire, Kindle Fire HD, Kindle Fire HDX, Fire HD 6, or Fire HD 7. The process is simple and will take only minutes.

Installing Snapchat to the Kindle Fire

1. Enable Installation of Applications from Unknown Sources
For the first generation Kindle Fire and Kindle HD, go to Settings > Device > Allow Installations of Applications

For the Kindle Fire HDX and Fire Tablets, go to Settings > Security > Apps from Unknown Sources

2. Open the Silk Browser and download the latest Snapchat installation file here.

3. Open the ‘Download’ folder and open the installation file to install Snapchat onto your device.

 

Since the last update, Snapchat now require Google Play Services. Follow our latest tutorial here to install Google Play Services and Snapchat: Get Snapchat for Kindle Fire
