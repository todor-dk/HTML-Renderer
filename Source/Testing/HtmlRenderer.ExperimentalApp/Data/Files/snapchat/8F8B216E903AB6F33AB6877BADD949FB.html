Snapchat on Computers

One of the most requested features for Snapchat is a Desktop (computer) version. There’s currently no official desktop application that will allow a Snapchat user to share an make snaps via his or her computer.

Wether Snapchat will implement this feature in the future is still unknown. The company seems to have a mobile centric development strategy and a Desktop app for Snapchat for either Windows 8 or Mac OS X is unlikely to fit in it.

Alternatives

However there are some unofficial alternatives to using Snapchat via a Desktop Computer. One of them is to use an App called BlueStacks. BlueStacks lets you run mobile applications on your computer and can be downloaded here.Â It runs both on Mac OS X and Windows computers. Once you’ve installed BlueStacks you’ll have to download Snapchat and run it from BlueStacks. Once you have it running you’ll be able to login and use your account.

Another option is to use a Mac app called Snapped. This app let’s you use Snapchat from your Mac to share and see all the awesome snaps that your friends are making and sharing. It’s currently in Beta so try at your own risk. Snapped is available for the Mac and can be downloaded here.
