        Formula to convert Celsius to Fahrenheit is:
        (°C x (9 / 5) ) + 32 = °F

        Therefore, for a negative (below zero) value of Celsius like “-12” the conversion will be:
        (-12 °C x (9 / 5) ) + 32 = 10.4 °F
