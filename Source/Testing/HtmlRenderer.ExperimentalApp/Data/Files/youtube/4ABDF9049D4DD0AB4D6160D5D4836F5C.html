YourTube

Open Source YouTube video downloader for mobile devices

What is YourTube ?

YourTube is a handy open source YouTube video downloader for mobile devices. Using YourTube, you can search for any video you like on YouTube and then save it in suitable format on your device.

YourTube is available for Android and Java ME environments.

Minimum requirements:

  • Android 4.1 (Jelly Bean) or later

OR

  • MIDP-2.0
  • CLDC-1.0
  • JSR 75 (File Connection API)
