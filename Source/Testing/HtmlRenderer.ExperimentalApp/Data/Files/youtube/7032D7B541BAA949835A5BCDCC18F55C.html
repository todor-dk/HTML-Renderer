To get started, you need the YouTube Downloader installation file. You can download it by going to the Home page and clicking Download.

After the download is complete and the installation file is saved on your computer, open it and begin the installation process.

Note: The installation is done in a few quick steps. Follow all the instructions and suggestions and the process will be complete within a few minutes.

The Setup screen will appear after you open the installation file. It will suggest you to close all running programs and applications before continuing the installation.

  • Shut down your browser and any other applications and click Next.
  • Select the suitable destination folder for YouTube Downloader by clicking the Change Destination Folder or continue as recommended – with the default folder (Program Files).
  • You will be directed to the License and Privacy Policy Agreement to read the information on copyright and other important aspects. Knowing and following the terms of Agreement is very significant for you and the company that developed the program, so make sure to read it carefully.
  • In order to proceed to the next steps of installation, check “I accept the agreement” box below and then click Install.
  • If you want to create a Quick Launch or a desktop icon of the program, check the corresponding boxes. In case you do not want any icons, do not forget to uncheck the boxes before clicking Next.

Note:If you didn’t close all applications, the Install wizard will notify you about any running applications and request your permission to close them automatically. In order to shut down the running programs and apps, administrator rights are required. Once the program is installed, the applications will be restarted.

  • After you click Install, another window will appear on the screen, displaying the installation progress. You will be informed about the successful installation by notification from the Install wizard.
  • Check the Launch YouTube Downloader box if you wish to run the program right now or the Finish button if you intend to use The YouTube Downloader later.

Note: It is recommended to restart your PC after installing the program.

Well done! You have installed YouTube Downloader on your PC! Enjoy your quick and easy music and video download!

Working with program

 

How to work with YouTube Downloader

 

You have installed a universal program which helps to find and download music and video from YouTube. Also with the help of YouTube Downloader you can listen to the favorite music online for absolutely free. Now we offer you to learn how to do it.

Minimalism is the key principle of YouTube Downloader. After the installation program will run automatically. If it doesn’t, run YouTube Downloader by using one of shortcuts or clicking the Quick Launch icon in the Start menu. The program window will open immediately.

You can keep the default settings or customize the program for your convenience.

 

How to customize YouTube Downloader settings

 

Select Edit menu — Language and choose one of the preset language packs.

To make it easier to find the downloaded music and videos, create a special folder on your PC. Then select this folder in the YouTube Downloader settings. Open the Edit menu, choose Change Destination Folder and add your preferred folder on the PC. Then click Select Folder. Well done!

Now you can set a default download format for music and video from YouTube. Click Edit - Default Download Format and select from the list of formats. Thus, you can download and convert content at the same time.

 

Working with YouTube Downloader

Now the program is customized and you are ready to search and download songs or video clips from YouTube.

 

Find your favorite song in a few seconds

The YouTube Downloader has two search fields: one at the top and one in the center of main window. Start typing the name of the song you're looking for, artist name or any line from the song. The program will display a list of tips. Click the Search button to start or press Enter on your keyboard.In case you want to find more than one song or video, open more tabs in the main window: just click Add New Tab and start typing in the search line.

 

 

Listen to the music online

With YouTube Downloader you can listen to music without registration. You don’t need to create playlists or download tracks on your PC. Right-click on the track opens the pop-up menu. Select Play and enjoy the perfect sound as long as you want. You can also use the Play button on the control panel at the top.

 

 

On the control panel you will find other elements:

At the bottom of the window you will find YouTube Downloader Repeat and Shuffle buttons for managing the tracklist.

Download music and video

Remember: with YouTube Downloader, you can listen to music and download files simultaneously. You can drag and drop tracks to Downloads Area for a download, while listening to your favorite track. It is necessary to specify the download format prior to the download by going to the Edit menu and choosing the Default Download Format setting.

Also, you can start downloading tracks by using the Download As in the track’s shortcut menu. You will also need to specify the desired format in the same menu.

If the download is successful, the State field will display the “Completed” status. In case of failure the State field will show you the “Failed” status.

Fast download feature allows you to upload videos in the following formats: .FLV, .3GPP, .MP4 and WebM. These formats are suitable for most portable media devices.
