Login Instructions
 

User Name: 

 Enter your user name (e.g. jsmith1234@na.convergys.com where na is your domain name)

Password: 

 Enter your current user password (password is case sensitive).

Note: 

 - User name is not case sensitive and no spaces are allowed.
- You must enter an @ sign between your user name and the domain name.
- Be sure to use your domain name based on your geographic location. Valid domain names include:
  • aspac (Asia Pacific)
  • emea (Europe)
  • india (India)
  • na (North America)
  • phil (Philippines)
  • sa (South America)
