Fonctionnalités de Windows Essentials

Le pack Windows Essentials 2012 comprend les logiciels suivants :
  • Messenger 2012,
  • Windows Mail 2012,
  • Windows Movie Maker 2012,
  • Galerie de photos Windows 2012,
  • Windows Writer 2012,
  • Microsoft SkyDrive 2012 ,
  • Windows Contrôle parental 2012 .
