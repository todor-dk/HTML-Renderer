Pokémon Go: APK, maps, tips, hacks and how to play from PC

With Twitter feeds blowing up around the world about Pokemon Go, at-home gamers may feel a little left behind. Fortunately, it is entirely possible to play Pokemon Go from a PC instead wandering around town to hunt down Pokemon. Basically, you have to download the Android phone emulator BlueStacks onto your PC. Afterward, you must install an app that helps you to fake your GPS location. Altogether, it takes about 20 minutes to get your PC ready to play. Although it lacks the functionality of real-world gaming, this system does work. Prepare for a few minor adjustments, and you can play Pokemon Go from your PC.

Getting Set Up

To play Pokemon Go on a computer using BlueStacks, you need to download the following tools:

  • BlueStacks
  • Lucky Patcher
  • KingRoot
  • Fake GPS Pro
  • Pokemon Go

For Pokemon Go to work, you have to make sure that everything else is downloaded first. Fake GPS Pro is used to spoof your location, and Lucky Patcher is designed to modify app permissions. Meanwhile, KingRoot is a root tool that is used with Android devices.

Installing BlueStacks and KingRoot

The first step is to install BlueStacks. You should go ahead and set up your Google account now if you do not have one. This will make it faster later on to connect to Pokemon Go. Once BlueStacks has been downloaded, simply install KingRoot. To do this, click on the AP symbol on the left side of BlueStacks. KingRoot will install automatically. Once it has been installed, scroll down to the bottom of the screen and choose the “Try It” button. Next, select “Fix Now.” Once your Security Index appears, choose “Optimize Now.” From this point, you can close KingRoot because you will not need it again.

At this stage, you should select the cogwheel in the upper-right corner of the screen for BlueStacks. Select the restart option for the Android Plugin. On the left sidebar in BlueStacks, open FakeGPS. This should make it automatically copy to BlueStacks. Afterward, you can close down the selection window.

Installing Lucky Patcher

Using the same install method within BlueStacks, install Lucky Patcher. Select APK on the left side of the window and install it. Once you are inside Lucky Patcher, go to the bottom right to find the “Rebuild” and “Install” buttons. From here, go to the sd card, Windows, and the Bst Shared Folder options. Choose the FakeGPS APK file. Install it as a system app and confirm your install. Afterward, you will have to reboot BlueStacks so that all of the changes are applied.

Setting Up Pokemon Go

Now that all of the other files have been installed, it is time to install Pokemon Go. Hold off on launching the app because it will not work quite yet. From BlueStacks, choose the cogwheel on the upper right and select “Settings.” From there, choose “Location” and make sure that you have changed the mode to high accuracy. You will also have to disable any Windows GPS services because these will prevent BlueStacks from spoofing your FPS. In Windows 10, you just have to select Windows key + I to reach the settings. From there, go to “Privacy” and “Location” to make sure that your device's location settings are turned off.

Setting Up and Running FakeGPS

For your game to work, your computer has to show a FakeGPS. Go back to Lucky Patcher and search for FakeGPS. Once you find it on the list, launch the app. Read through the instructions about how to operate it before closing the instruction window. There are now three dots on the upper right. Choose the one that is for the settings and select “Expert Mode.” Use the back arrow on the upper left to return to the map. When the map appears, pick the location that you want to be at. You can save the location to your favorites or choose another location.

Get Ready to Play!

Now that you have downloaded everything, you are ready to start playing Pokemon Go. From this point, you can set up Pokemon Go like you would on a typical Android device. Log in through Google and the app will automatically load the account that you set up. Once the game begins, it should automatically go to the fake location that you chose. If you want to choose a different spot, you have to return to FakeGPS and choose a new location. Try not to do long-distance jumps because Pokemon Go will think that you are spoofing your GPS.

When you find your first Pokemon, the system will realize that your camera does not work. It will ask if you want to turn off augmented reality (AR) mode. Choose the “Yes” button, and you can continue to play Pokemon Go.

Tricks and Cheats

Spot the Rustling Grass

In normal Pokemon games, you would not bother with any long grass because you could be attacked by Rattata. With the new version, rustling grass is an indicator that a Pokemon could be nearby. If you see rustling grass, check it out.

Go to Populated Areas

Populated areas are the best place to go to catch Pokemon. Parks are ideal because you do not have to worry about running out of Poke Balls due to the many PokeStops. Finding different terrain is also a good idea because you will gain access to different types of Pokemon.

Check Out Parking Lots

For some reason, Pokemon seem to like spawning in parking lots. Find a parking lot that is near a PokeStop, and you should easily be able to see multiple Pokemon.

Leveling Up Quickly

When you reach level nine, you will be given a Lucky Egg. This egg can double the XP you get for 30 minutes, so use these minutes wisely. Ideally, try to get as many Pidgeys as you can and keep them in your pocket. When you have a number of Pidgeys and Pidgey Eggs, evolve them using the Lucky Egg. While you would normally get 500XP for evolving a Pidgey, the Lucky Egg will mean that you get 1000XP for the Pidgey. Plus, you only need 12 Pidgey candies to evolve it.

Get Incense and Lures

While Pokeballs help you capture new Pokemon, make sure to stock up on incense and lures. Incense will lure Pokemon to your location for 30 minutes. Meanwhile, the lure module will bring wild Pokemon to the PokeStop for 30 minutes. When you go by a PokeStop, look for falling pink petals. These petals show that someone has left a lure module in place, so you can take advantage of their lure for up to 30 minutes.
