《十二公民》：中国会有陪审团制度吗？

作者：邓冬辰来源：蝌蚪五线谱发布时间：2015-05-21
《十二公民》是向美版和俄版《十二怒汉》致敬的片子，其核心陪审团制度，能否在中国存在呢？

 

　　导演： 徐昂

　　编剧：李玉娇 / 韩景龙 / 徐昂

　　主演：何冰 / 韩童生 / 钱波 / 赵春羊 / 米铁增 / 更多...

　　类型：剧情 / 悬疑 / 犯罪

　　制片国家/地区： 中国大陆

　　语言：汉语普通话

　　上映日期： 2015-05-15(中国大陆) / 2014-10-19(罗马电影节)

　　片长：106分钟

　　又名：12公民 / 十二个中国人 / 12 Citizens

　　5月15日在大陆观众见面的《十二公民》，早在上映之前就吸引了我的眼球。这不光是因为它的前身多么有名——美版《十二怒汉》简直家喻户晓，俄版的翻拍也各有千秋；也不是因为它去年在罗马电影节上斩获了“马可·奥雷利奥奖”，而是因为，这部电影引发了我一个思考：中国会有陪审团制度吗？

　　疑问从何而来？

　　《十二怒汉》的主要剧情是：12位陪审员就一位小男孩弑父案作出讨论，决定那个小孩是有罪还是无罪。其中核心要素无疑是陪审团制度的存在。而刚刚上映的《十二公民》里，时空变幻，中国这片土地上有一起富二代弑父案，社会舆论纷纷扬扬，一个由学生、家长组成的模拟法庭，将要对此案作出评判。

　　发现问题所在了吗？模拟法庭！没错，就是这个，中国不像美国，有陪审团制度，故而中国版十二怒汉只好耍了个滑头，不但把真实法庭变成模拟法庭，还让检察官混迹其中，以达到影响真实判决的效果。好吧，这也算苦心孤诣了。

　　中国目前没有陪审团制度，那么未来会有吗？

　　在回答这个问题之前，先让我带领大家走进陪审团制度的前世今生。

　　11世纪的欧洲，正如现在人们所熟知的一样，是一个公国林立，贵族争相裂土分疆，而教会影响无处不在的黑暗时代。当时在法国的诺曼底，有一位贵族被分封于此，这人是诺曼底公爵，威廉一世。此人窥视英国国王之位已久，又有灵活的外交手段和强大的野心；相比之下，当时的英国国王哈罗德，却是一个守旧、不懂外交之人。威廉合纵连横，成立了欧洲反哈罗德联军，在黑斯廷斯与哈罗德决战。哈罗德在混战中中箭身亡，冷酷而狡诈的威廉终于坐上了英国的王位，开启了英国历史上的诺曼王朝。

　　诺曼征服为英国带来了陪审团制度的雏形。我们通常认为现代意义上的陪审团制度发源于英国，但回顾历史，显然，我们不要忘了法兰西文化在这之中也起过重要的作用，是必不可缺的一环。

　　现在的陪审团制度，分为大陪审团和小陪审团两种。

　　大陪审团，又称 “起诉陪审团”，其角色大致相当于“人民检察院”。它通常由案发所在地区的23位普通公民组成，其职能并非裁定被告是否有罪，而是在法院正式立案之前，传唤与案件有关的人证和物证，决定对有争议性的重大案件是否应当立案起诉。一旦作出裁定，大陪审团就自动解散，由检察官接手办案。在“拉链门”案中，把克林顿折腾得灰头土脸、狼狈不堪，并裁决立案起诉的陪审团，就是这种大陪审团。

　　另一种是小陪审团，又称“审判陪审团”，其角色大致相当于“人民法院 ”。它通常由案发所在地区的12位普通公民组成，负责对刑事和民事案件中的被告人是否构成刑事犯罪或民事侵权作出裁决。如果裁决无罪或不构成侵权，审判即告结束。如果裁定有罪或侵权成立，则由主持审判的法官依法量刑，作出正式司法判决。在球星辛普森涉嫌杀人案中，以检方呈庭证据漏洞百出为由，将这位橄榄球巨星无罪开释的陪审团，就是这种小陪审团。

　　而电影中出现的，就是后者，12人组成的小陪审团。

　　如果一言蔽之来说明陪审团制度的特点的话，那就是“外行领导内行”。

　　这种制度在欧美尚有所争议。反对者认为，由一帮外行组成的陪审团，竟然高居于职业法官之上，对重大法律问题做出最终裁决，这简直愚蠢透顶。我们把这种说法称之为“司法民粹主义”。相对的，支持方认为这体现了司法民主，是具有优越性的。

　　但我认为，这中间的对对错错，实际早有定论，并不存在争执。

　　桑德尔讲正义，举例子说，你是医生，可以选择救五人，也可以选择救一人，但你只能救一边。几乎所有人都会选择救五人。改变条件，有五人等着救命，你可以选择杀一人救这五人，也可选择不救。所有人都会选择不杀人。看起来，似乎正义没一定之规。

　　其实，上述两种选择背后的逻辑是一致的：尊重生命、自由与财产等自然权利。同样是救五人，情形一不侵犯后一人的权利，情形二则侵犯无辜者的权利。可见，普通人对自然法/自然权利有着朴素而强烈的认知，人同此心，心同此理。陪审团制对此充分尊重并善加利用，足见高明。

　　那陪审团制度在中国的发展情况如何呢？并不是没有尝试。

　　2004年8月，全国人大常委会通过“关于完善人民陪审员制度的决定”，这似乎是一个示好信号。但这说的陪审员，偏向于大陆法系的陪审员，需要任命，有五年的任期，实质上是名“准法官”，和咱们理解上的陪审团不太一样。

　　目前看来，从陪审员，到陪审团还有一定的距离，这是什么导致的呢？

　　陪审团制度对陪审员和规则有严格要求，是高度讲究规则的。在审理中，法官会不断给陪审团发出各种指示，如指出某项证据是非法证据、不能采信。如果采信了非法证据，那陪审团的裁决就无效。如果一个社会不能照规则玩，那陪审团制就很难成功运作。

　　敢不敢用陪审团，能不能用好陪审团，其实考验的是司法职业共同体的自信。当我们高声叫嚷着陪审团可以“还人清白”的独立性的同时，不要忘记富有经验的律师们操控舆论影响陪审团员们的能力。若要这种操控不能得逞，就需要同样“老到、辛辣”的法官与之抗衡。说到底，这是对我们司法从业人员，还有公民一个共同的考验。

　　陪审团制度是个好制度，只要公民素质持续提高，司法从业人员的水平不断提高，司法环境逐渐的改善，那么未来中国，必定会有陪审团制度的一席之地。
