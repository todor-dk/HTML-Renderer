Installation instructions

Microsoft Windows

  1. Extract the Bezier.scr screensaver file to your Windows folder (usually located at C:\Windows) to install the screensaver.
  2. If you install it to a different folder, or the screensaver does not appear in the Screen Saver Settings panel, locate the file where you extracted it, right-click it and select "Install".
