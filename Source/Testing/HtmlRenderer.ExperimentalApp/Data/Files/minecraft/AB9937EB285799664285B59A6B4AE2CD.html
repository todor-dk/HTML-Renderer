Análisis detallado

Por Francisco Braña

Minecraft es un juego muy popular que combina construcción, plataformas y acción y que pone a prueba tu capacidad creativa para crear tu propio mundo. En Minecraft te encuentras ante un mundo por construir y su evolución depende sólo de ti. Tendrás que recoger recursos para poder edificar y sobrevivir día tras día, ya que durante la noche monstruos tan... "molestos" como los Creepers te acecharán.

El aspecto gráfico es bien simple, pero es una parte característica del juego. Todo se forma a partir de cubos: puentes, ríos, montañas, túneles, edificios, bosques... y forma de cubo tendrán también los materiales con los que construyes. Si el aspecto "retro" del juego te echa para atrás, merece la pena que busques en la web un paquete de texturas a tu gusto, con el que de forma muy sencilla cambiarás totalmente los gráficos y el diseño artístico del juego.

Según tus preferencias puedes jugar en modo "supervivencia" y lograr defenderte en un mundo hostil, o bien en "creativo", donde podrás volar y tener cualquier objeto a tu disposición para así crear las construcciones más imponentes. Minecraft es un juego que al principio parece sencillo, pero que goza de gran profundidad conforme avanzas y ofrece grandes posibilidades. Gracias al modo multijugador que ofrece además podrás disfrutar de lo lindo con tus amigos y jugar en servidores con cientos de personas.

Con Minecraft tienes muchos juegos en uno, y sobretodo una total libertad para elegir como quieres divertirte. No dejes de estar atento a este juego, ya que constantemente se añaden nuevas características gracias a sus actualizaciones.

Requisitos

- Procesador a 1.3 Ghz
- Memoria RAM de 2 Gb.
- Tarjeta gráfica con 256 Mb.
- Espacio en disco duro de 90 Mb.
- Registrar una cuenta en la web de Minecraft (el proceso es gratis).
- Java Runtime Environment 6 o superior.

Limitaciones

- Si no tienes cuenta Premium (de pago), solo podrás jugar 5 dias (de juego) por cada mundo creado.

Nota

Para hacer correr Minecraft necesita una serie de requisitos ya que se trata de un programa de java, por eso tu extensión es jar: minecraft.jar
Necesitas tener instalada una versión de Java, es suficiente con la versión el jre (java runtime enviroment). Para saber si ya lo tienes instado puedes utilizar el comando "cmd" sin comillas en una ventana del DOS y posteriormente utilizar el comando "java –version" sin comillas y con un espacio entre java y el guion.
Si ya tienes instalado Java no hay problema, en caso contrario puedes descargarlo aquí para Windows y aquí para Mac.
Una vez tengas Java instalado tienes que utilizar el comando java -jar archivo.jar desde la consola del DOS para ejecutar el juego.
