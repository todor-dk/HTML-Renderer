[bd] Mails 1.0.1

Make sending email easier.

  • Overview
  • Updates (8)
  • Reviews (36)
  • Version History
  • Discussion
  1. Compatible XF Versions:
    • 1.1
    • 1.2
    • 1.3
    • 1.4
    • 1.5
    Visible Branding:
    No
    [FEATURES]
    1. External email provider: Amazon SES, Mailgun, Mandrill, SendGrid
    2. Bounce Processing: Amazon SES, Mailgun and Mandrill

    Demo: https://xfrocks.com/

    [INSTALLATION]
    1. Upload files and directories inside `upload` to XenForo's root directory. Select "Merge" for everything if you are asked.
    2. Import the .xml file using XenForo add-on importer.
    3. Configure provider as needed.
