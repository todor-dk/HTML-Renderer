El proceso completo para formar el número de telefóno de destino es:

  • 1. Código de salida nuestro país (En caso de existir alguno)
  • 2. Código telefónico del país destino
  • 3. Código de la ciudad destino
  • 4. Número de teléfono al que se desea llamar.
