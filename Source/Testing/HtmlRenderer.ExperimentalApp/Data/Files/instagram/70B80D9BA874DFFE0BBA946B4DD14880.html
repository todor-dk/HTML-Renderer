Instagram without a smartphone

Foto Rulez – is an application for image processing using a webcam. You can do your photos funny and lovely using a lot of effects. For example users can sit in the front of a camera and make funny faces like in the room with the false mirrors. For those who wants something more there are a lot of filters that can mimic various photographic effects. This application can also process your uploaded photo. But the main purpose of Foto Rulez is work with a webcam. User can take a photo at any moment using the selected effect. The result of processing can be easily saved on your PC. Instagram your photos online. It is easier than Photoshop, funnier that Instagram.

  • Once the application is loaded, allow access to your webcam.
  • Choose an effect you like by using "left" and "right" buttons.
  • Take a picture by using the button with photo camera image on it.
  • You can save the resulting image to your PC by clicking the save button.

You can also switch to russian version of Foto Rulez

To use this application you should install the latest version of Adobe Flash Player.
