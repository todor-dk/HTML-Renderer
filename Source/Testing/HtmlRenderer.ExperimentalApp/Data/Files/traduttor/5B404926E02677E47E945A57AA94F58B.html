I have worked around the clock for a few weeks now to make a new version before December 1. But now forced to realize that I do not have time to get all the new features completed.

So for those of you who do not want to wait any longer I release this version (2.9.23) where not everything that comes in version 3.0 is available. In this edition English is the only available user interface language. This is because I'm about to change the structure of the language file, so that it will be easy to manually edit it after a version update.

 

In this version, you can at least start translating with two new translators.

 

 

Google Translate

  • Google's free translation service "Google Translate", available at translate.google.com. This software is only an interface to Google online translation service and I am not associated with Google translation service in any other way. Google can ban IP address of users who use their translation service excessively and can result in temporary or permanent non functioning of this option.

Google Translate API V1
  • Google Translate API v1 was officially deprecated on May 26, 2011; it will be shut off completely on December 1, 2011.

Google Translate API V2
  • Replaces V1, this is now available as a paid service. USD 20/1,000,000 characters. You need a "Keycode" first to use this service.

Microsoft Translator API V2
  • With the "Microsoft translator" you can translate 2,000,000 characters for free every month, you need a "AppID" first to use this service.

To get a Keycode and/or AppID, you find links i the Translation Settings form. Menu "Tools - Translation Settings"

Some of the menus is disabled in this version, but will be available in the next. I hope that the program is not too buggy, I have not had time to test it enough. Please send me an e-mail if you find anything wrong, so I can correct this to version 3.0.

In the next version the user interface be available in all languages as before, and the ability to create your own language file, if your native language is missing.

/Pappsegull Sweden
