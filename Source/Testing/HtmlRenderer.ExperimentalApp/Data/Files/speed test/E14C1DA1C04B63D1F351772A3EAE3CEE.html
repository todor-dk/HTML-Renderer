Website Speed Test Terms

Request Methods

  • GET: Requests data from a specified resource
  • POST: Submits data to be processed to a specified resource

HTTP Status Codes

  • 2xx: The server responded with a successful code
  • 3xx: The request was redirected to another target
  • 4xx: A client error occured, for example 404 page not found
  • 5xx: A server error occured, for example 500 internal server error
  • Error: Connection error, no response from the server

File Types

  • text/html; charset=UTF-8: A web page or post
  • text/javascript;charset=utf-8: Javascript file (file.js)
  • application/x-javascript; charset=UTF-8: Javascript file (file.js)
  • Javascript file (file.js)
  • CSS file (file.css)
  • text/css (external CSS file)
  • image/png: PNG file (file.png)
  • image/jpeg: JPEG file (file.jpeg)
  • image/gif: GIF file (file.gif)

Timeline

  • Wait: The web browser is waiting for data from the server
  • Receive: The web browser is receiving data from the server
