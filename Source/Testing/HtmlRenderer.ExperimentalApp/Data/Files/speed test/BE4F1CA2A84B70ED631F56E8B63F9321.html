Internet Connection Speed

How to test your Internet connection speed

To test your connection speed, click on this link: http://speedtest.tds.net. 

A new screen will appear. (Click on the button labeled "Start" on the new screen.

The test will run for a few minutes, and then a new screen will appear with your speed test results. In the box below the phrase “You may want to copy/paste this box:” you will see 2 numbers in bold. The first number is your download speed and the second number is your upload speed (see screenshot above).

If you had a 1.5Mb DSL connection the maximum number that you should see is about 1536Kb. Your results may be up to 20% lower due to the overhead required in the transmission of information to the internet. So if you had a 1.5Mb DSL connection you should see a download speed around 1228.8Kb per second.

Please remember the test results from the speed test site can be skewed by many factors. A few of those factors can be found thru our frequently asked question: What can cause slow internet speeds for DSL?

You may need to get the latest version of the Java Virtual Machine at http://www.java.com/en/download/index.jsp. Please note that TDS does not provide or support the Java Virtual Machine.
