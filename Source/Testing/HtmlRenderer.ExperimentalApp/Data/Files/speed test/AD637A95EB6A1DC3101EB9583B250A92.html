Speed Test

  • Home
  • Support
  • Speed Test

The PenTeleData speed test is a diagnostic tool that is designed to help our Technical Support staff evaluate the performance of your Internet connection within the PenTeleData network. In order to run the PenTeleData speed test, you will need to have the latest version of Java installed. If you do not see the java applet below, please visit www.java.com and follow the on screen instructions.

If you have a commercial account and feel that you are experiencing slow data transfers on your Internet connection, you can contact PenTeleData’s Network Control Center at 1-877-4PTDNCC (478-3622). When contacting the Network Control Center, please be sure to reference the "Test Result #XXX" which can be found just below your test result speeds.

To start the test, click the button labeled "Click here to begin your Test Speed." Your test results will be displayed momentarily.

NOTE: In order to achieve better speed test results, please ensure that you take the speed test on an average of three times by using the “Refresh/Reload” button within your web browser or by using keyboard shortcut keys. Internet Explorer users, select the “F5” key along the top of your keyboard. Firefox users should press the "Ctrl" key and then, choose “R” on the keyboard (Crtl+R).
