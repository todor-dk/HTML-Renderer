Speed Test ทดสอบความเร็วอินเตอร์เน็ต

บริการทดสอบความเร็วอินเตอร์เน็ตของคุณ สำหรับทุกท่านที่
ใช้งานอินเตอร์เน็ต ไม่ว่าจะเป็น ADSL Hi-speed internet,
Dial up internet หรือ Broadband internet สามารถใช้บริการ ทดสอบความไวเน็ต ของ 3BB Speed test ได้เลย
ซึ่งบริการให้ทดสอบความเร็วเน็ต เป็นบริการฟรี ที่ 3BB จัดทำระบบ Speed test ขึ้นมาเพื่อผู้ใช้งานอินเตอร์เน็ตทุกราย
The Ookla Speed test requires at least version 8 of Flash. Please update your client.
  

ข้อแนะนำในการใช้บริการ 3BB Speed test

  • สำหรับการให้บริการทดสอบความเร็วเน็ต (Speed test)
    นั้นอาจจะไม่เท่ากันในแต่ละช่วงเวลา
  • ผลที่ได้จากการใช้ทดสอบ Speed Test นั้นอาจจะไม่ตรงกับความเป็นจริงได้ หากมีการเปิดใช้งานอินเตอร์เน็ตในรูปแบบอื่นๆ เช่น ดาวน์โหลดข้อมูลในขณะทดสอบ
  • ในการใช้เช็คความเร็วอินเตอร์เน็ต ควรทำ Speed test หลายๆครั้ง
    เพื่อหาค่าเฉลี่ยที่ใกล้เคียงมากที่สุด
  • การเช็คความเร็วเน็ตด้วยระบบ 3BB Speed test นั้น เป็นเพียงข้อมูลส่วนหนึ่งในการตรวจเช็คความเร็วอินเตอร์เน็ตเท่านั้น มินำไปอ้างอิงทางวิชาการได้
