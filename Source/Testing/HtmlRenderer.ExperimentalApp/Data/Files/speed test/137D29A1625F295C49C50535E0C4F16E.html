What is firewall check?

Testing for the presence of a firewall or NAT device between the client and the server is done using both TCP (Transmission Control Protocol) and UDP (User Datagram Protocol) traffic. This feature can be completely disabled in the application if preferred. Display the results as a detailed view or a simple summary.


What is latency test (ping and jitter)?

We measure latency over HTTP. The Line Quality Test instead uses a direct TCP socket for results much closer to that of an ICMP Ping. Jitter is also calculated as the variance over the series of latency measurements.


What is packet loss test?

The test sends many UDP packets from the client to the server and determines how many of them arrived. This packet loss percentage is a key metric for quality in real-time network applications such as VoIP.


How to testing your internet speed?

Make sure that you are not using your internet connection for anything else while the test runs.
