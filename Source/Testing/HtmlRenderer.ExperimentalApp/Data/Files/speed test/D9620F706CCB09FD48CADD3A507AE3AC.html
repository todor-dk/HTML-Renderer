Speed Test

Posted by Max Lehmann on November 14, 2013
Tweet

What is Speed Test?

Speed Test is a potentially unwanted program. It is usually part of another freeware setup. Speed Test is an application that allows you to check your current upload and download speed. Although on the surface there is nothing illegal about Speed Test, the fact that it comes bundled with freeware raises a few questions about this application’s reliability. Therefore, if it was not your intention to have Speed Test on your computer, it would be for the best to remove Speed Test via your Control Panel right now. Also, remove all the other related programs while you are at it.

Where does Speed Test come from?

Security experts claim that Speed Test may come bundled with various freeware applications and adware. However, most of the time this application travels with Codec Performer Update. This update is being “recommended” by Codec Performer Update is Recommended pop-up (fake alert), which claims that you need to install Codec Performer Update in order to support all the possible media formats. Technically, the only thing that should be installed on your PC is FFDsShow and Haali splitter codec libraries.

However, apart from the actual codec libraries, you also end up with Speed Test, PC Performer and Free Games freeware applications on your computer.

What does Speed Test do?

It is true that quite a lot of codec installations come with additional software. However, most of the time, if the code update is legal; users are given the option to choose whether they want to install additional software or not. That is not the case with Codec Performer Update, because Speed Test is installed on your computer automatically.

Once the installation is complete, you are redirected to performersoft.com website, where you are offered to run an internet speed diagnostic test immediately. Since Speed Test comes coupled with PC Performer application, if the app displays a low internet speed, then you would be offered to utilize PC Performer to optimizer your PC performance immediately.

How to remove Speed Test?

Apart from the fact that it checks the Internet speed (online!), virtually there is nothing useful about Speed Test. What is more, it is bundled with other freeware applications that are barely useful, so there is no need to have Speed Test or PC Performer on your system, as they only take space and system resources.

In order to terminate Speed Test, you can go straight to Control Panel and uninstall it at once. It would also be good idea to terminate all the other related applications, as they might be related to potentially unwanted programs. What is more, after manual removal, you should perform a full system scan with SpyHunter free scanner, as there may be some unwanted files left. Do all it takes to make sure your computer is clean.

Uninstall Speed Test

Windows 8

  1. Move mouse cursor to the bottom right of the screen.
  2. When Charm bar appears, click Settings and go to Control Panel.
  3. Open Uninstall a program and remove Speed test, PC performer and Free Games.

Windows Vista & Windows 7

  1. Open Start menu and click Control Panel.
  2. Go to Uninstall a program and remove Speed Test, PC Performer and Free Games.

Windows XP

  1. Open Start menu and go to Control Panel.
  2. Select Add or remove programs and uninstall Speed Test, PC Performer and Free Games.
