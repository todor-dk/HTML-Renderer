Use WhatsApp Web Client With iPhone: Step-by-Step Guide

By Nageen Zahra  /  2 years ago Share Tweet


As always, Apple is taking its time to thoroughly examine the WhatsApp Web Client before officially adding it to the App Store for downloading. However, we have a tutorial for all iPhone users to enable WhatsApp Web Client on iOS devices.

It is pretty simple: the WhatsApp Web Enabler allows iOS users to make use of the Web Client on their devices as it works just like the Client. However, the tweak is only available for Jailbreak iPhones. So, what do you have to do? Learn that in the following steps.

How To Use WhatsApp Web Client With iPhone

Step 1:
First thing’s first, you have to install the WhatsApp Web Client tweak on the iOS platform.

Steep 2:
Once you have done that, a ‘WhatsApp Web’ icon or button will automatically appear in the Settings menu of your chat app. Tap the button and turn on your computer for the next step.

Step 3:
Then jump to the website web.whatsapp.com and scan the QR code on the home page of the website.

WhatsApp Web Enabler also allows you to manually enable or disable the option from the Settings menu of your app in iPhone. The updated version of the tweak comes with some bug fixes to instant crashing reported by users.

WhatsApp Web Client comes in handy if you use laptop/ desktop a lot. It is an interactive and convenient way of staying in touch with friends while working or browsing online.

The service is currently only limited to Android for now. However, we are sure it will hit iOS soon, but keeping in mind Apple and its in-depth examination for malwares before approving the app for downloading, it might take a while.

You can download WhatsApp Web Enabler from Cydia’s BigBoss repository. It is available for free downloading and supports iOS 4 till 8.x on iPhone.
