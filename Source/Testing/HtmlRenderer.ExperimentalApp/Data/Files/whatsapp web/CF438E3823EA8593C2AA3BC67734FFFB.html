How to use Whatsapp Web

Kevin John
January 22, 2015
Android, Featured, Web
1

Following the ban wave  of Whatsapp Plus users Whatapp CEO Jan Koum posted a status update announcing Whatsapp for Web.

 

 

The update is not yet live on the Play Store but you can grab the beta from here and install to start using Whatsapp for Web.

http://www.whatsapp.com/android/beta/WhatsApp.apk

 

Once you have installed the beta Whatsapp you will need to go to

https://web.whatsapp.com/

Now open whatsapp go to menu and tap on Whatsapp Web and scan the QR code you got on web.whatsapp.com

you should be logged into Whatsapp Web now.

The popular messaging service also said that users’ phones need to stay connected to the Internet in order for the web client to work, and they must install the latest version of WhatsApp on their phone.

Because of “Apple platform limitations,” the web client is not currently available to iOS users, WhatsApp said. It did not address when or if that would change.
