WhatsApp Web Version in the Works – App for Desktop Might Launch Soon

By Rafia Shaikh
Dec 16, 2014
16Shares
Share Tweet Submit

WhatsApp helps more than 600 million users to connect with each other. Largely used on smartphones, we showed you how to use the world’s most famous messaging app on tablets too. While there is also a workaround to get the app on desktop, it’s neither officially supported nor is a very ideal solution for messaging. However, latest reports indicate that we might soon see WhatsApp web version.

WhatsApp web version launch:

WhatsApp has largely remained mobile-focused for years now. If latest rumors are anything to go by, this might be changing as WhatsApp web version launches. We have so far never seen any speculation around a web version of this popular messaging app. Pavel Durov, co-founder of another messaging app Telegram reported that WhatsApp has tried to hire its web developer.

That is not the only thing indicating towards a web presence of WhatsApp though. Discovered by the folks at AndroidWorld.nl, a recent update rolled out to WhatsApp carried various mentions of WhatsApp Web and ways to log on to web service along with other services. Presence of efforts of creating a web version of the app adds into the weight of rumors around the matter.

WhatsApp won’t be officially announcing or confirming these rumors unless the day it is ready to roll out the web offering of the app. However, looking at our readers’ requests of having ways to get WhatsApp for desktop, we believe the web offering will be embraced quite positively by users. What are your thoughts?

– Source: Android World
