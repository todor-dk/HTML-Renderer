Use WhatsApp Web Version On Any Web Browser

July 26, 2015 by Sumit Khantwal Leave a Comment

After acquisition of WhatsApp by Facebook, WhatsApp has come up with  few changes, though BlueTick was criticised by many users. But few month back they came up with the much awaited feature, at least for me it is. Yes the web interface of WhatsApp is live now.WhatApp Web is officially available for Windows and MAC PC. Yes it works on Google Chrome, FireFox for windows and safari for MAC and soon Microsoft edge will support it too. Earlier it was available only for chrome but now it supports every browser and PC.

Using this Web version you will be able to use WhatsApp on Windows PC and MAC. Initially we were using a particular software which we call  Android Emulator. BlueStack and YouWave are the best example which we generally use  for this purpose, but now task is much more simple.

No need to install those hardware resource intensive emulator. WhtatsApp web has made it simple and clean.User-friendly interface and ease with which it could be used will make you a fan.

WhatsApp Web Version For Windows PC And MAC

The best part is that we don’t need to install any particular software.It will work on basic platform which every one of us usually have.What we need is latest version of WhatsApp and browser of your Choice.Like, Google Chrome and FireFox for Windows PC and Safari for MAC. If you don’t have latest version then download now.

  • Download Latest version if you don’t have one.

Download Now

Keep in mind it will work only on Google Chrome.We tried with FireFox and got this instruction.

WhatsApp Web New Update : Edge Browser Support

WhatsApp Web now works with Firefox for Windows and Safari for MAC and Microsoft edge will support it soon . Initially it was made available for chrome only. MAC users, Finally wait is over.

Use WhatsApp Web On Browser

After installing latest version, you need to move on their Web platform.You can access this web platform using browser. Below is the landing page, Page consist of list of supported devices and QR code.This QR code you need to scan from your mobile.

  • Read Dual WhatsApp Accounts On One Device

Step 1 : Open WhatsApp App on your Smart Phone and locate WhatsApp web.This function is grouped with [ Status and Setting ]. Step 2 : After Tapping on screen, on screen camera will start working.Scan the QR code of browser with the help of  this scanner.

As Soon as code is read by mobile it will Sync mobile device with computer and one can easily use WhatsApp on windows and MAC PC/laptops.

Note :- This Feature doesn’t install WhatsApp on computer,it simply Sync it with computer so one need to have active internet connection on mobile device too for proper messaging

Though this feature is Available with almost on all platform.But for convenience have a look on list on devices.

Supported Device :

  • Android
  • Windows Phone
  • Blackberry
  • Blackberry 10
  • Nokia s60
  • Nokia s40

Unfortunately we don’t have IOS on this list. This is due to limitation impose by Apple on its platform. Sorry IOS user I can feel your pain 🙁

Though people tends to love smaller devices for messaging purpose but for those who are too busy with their computers this WhatsApp web feature is a great way to enjoy messaging.

Now enjoy Web version on Windows and MAC without any problem
