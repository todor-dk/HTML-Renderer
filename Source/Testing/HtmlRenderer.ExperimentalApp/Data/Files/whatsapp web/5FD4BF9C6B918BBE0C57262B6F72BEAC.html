How to Sign in WhatsApp Web Online?

In order to get over with this vey special way to use WhatsApp on computer, following are the instructions you need to go through. It will not take any longer than 2 minutes.

  1. Update the WhatsApp application which you are using on your smartphone. Make sure you are using the latest version first.
  2. Launch the Chrome or Firefox browser on your computer or laptop. Open this website URL for WhatsApp Web.
  3. Once the website is opened you will see a QR code. Now, you need to pick your phone again.
  4. Move into Options/Settings of WhatsApp application and there you will see an option with name ‘WhatsApp Web’. Tap on that!
  5. It will launch the camera module of your phone then and your job is to scan that QR code using the rear camera. Yes, try taking a picture of that QR code using the camera module within WhatsApp.

Done! The login will take place and instantly you will see all your conversations, contacts and profile running right within the Chrome or Firefox, whichever browser you are using at your end.
