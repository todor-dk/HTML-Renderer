Hidden Features of WhatsApp

The known features that we will discuss are officially explained and used by all users across the world, but there are several tricks or hidden features in the WhatsApp Android App which are applicable in some devices. We are going to disclose these hidden tools or features in the App here.

For our readers especially, here are some of the WhatsApp hidden features as stated below;

  • BOLD and ITALICS fonts: In almost all WhatsApp text box, there is no bold or italics option for text formatting. Although some messages that we receive are in bold and italics. The user cannot do this by simply typing in the word document in bold or italics format and then paste it in the WhatsApp text box. The message will still be normal text. So, to get through this you have to do the following:
    • For Bold texts: type * symbol before and after the text eg: *Hello* and you will get “Hello“
    • For Italics: type _ symbol before and after the texts eg: _hello_ to get “hello“
  • Data saving: Many users do not know that you can save access flow of data from the app as it provides a filter o the network when not in use. To initiate this option, follow these steps:
    • On your WhatsApp Application, got to “Settings > Data usage”
    • From there, users can control the auto-download feature for different media messages when connected to different network connections. The user can also switch the “Low data usage” option on to revert less data consumption while attending WhatsApp calls.
  • Star Messages: Many users still cannot figure out how to star a message and what is the use of the star messages. Sometimes, some messages like a contact or address details can be sent to you and when you prolong the chat, you cannot find the message from older messages. The importance of the star message is to retain it whenever you want to see that information without having to scroll the entire chat for it. To star a message simply:
    • long press on the message that you want to star.
    • The message will highlight and new options will emerge on the top of the screen.
    • Select the star looking icon to mark the message.
    • You will see a small star on the message.
    • To view the starred messages simply click on the name of the chat and scroll down to see all the starred messages in the information.
