WhatsApp Web

Whatsappen.comWhatsApp Web


The web client ‘WhatsApp Web’ is introduced in January 2015. WhatsApp Web can be used in combination with all by WhatsApp supported phones. 

WhatsApp Web is an extension of your phone and mirrors the conversations and messages from your mobile device. All your messages and shared media appear on both your phone and web browser.

To connect to WhatsApp Web click on below ‘Open WhatsApp Web’ button. Scan the QR code with your mobile phone (WhatsApp=>Settings=>WhatsApp Web) and start chatting!

Please note that your phone needs to stay connected to the internet in order to work with WhatsApp Web.

Supported browsers WhatsApp Web

WhatsApp’s web client can be used with Google Chrome, FireFox, Opera, Safari and Microsoft Edge.
