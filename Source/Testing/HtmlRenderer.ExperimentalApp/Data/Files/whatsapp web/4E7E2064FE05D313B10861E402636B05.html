Unrelated to Whatsspy - WhatsApp in browser without a phone connected

Since a few months there is the WhatsApp web-app for use in a Browser. But actually the web-app is just an extension of your physical phone, messages sending and receiving is still entirely done by WhatsApp on the phone. So without the phone the web-app doesn't work.

But I thought, we have WART and we have WhatsSpy. Proving that a phone is actually not nessesary to run WhatsApp. When WhatsSpy has the login info for the WhatsApp server, it can run entirely on a PC (or Linux/RaspberryPi etc.) without the phone.

My idea was: Is there a way to adapt this code to use it for WhatsApp web-app? So a WhatsApp program for PC which serves as the WhatsApp phone client. The goal would be to use the WhatsApp web-app with just the program in background and without a phone connected.
