    WhatsApp Web

    Posted By hareshperiyasamy on Jun 27, 2017 in WhatsApp Desktop, WhatsApp for PC, WhatsApp Web

    WhatsApp, the biggest messaging platform has finally hit the computers with its brand new feature WhatsApp Web. The WhatsApp users have always been waiting for such an option for using WhatsApp on PCs. It is indeed a great experience to chat on the big screen comfortably with your keyboard and mouse. With a huge crowd of 320 million active daily users, WhatsApp Web is a complete treat for all. You can use WhatsApp everywhere and almost with any device; now the list has extended further with WhatsApp Web. WhatsApp Web is very simple to access as it does not require any login mail id or password rather uses highly secured QR Codes. The interface has all the features similar to the WhatsApp App.

    WhatsApp Web

    Now use WhatsApp while you work from office! from home flexible with WhatsApp Web. This Web client of WhatsApp is very responsive with the computer screen so that you can access across PCs and tablets. WhatsApp uses end to end encryption for making it difficult for anyone to interrupt between conversations. WhatsApp Web works fine with Chrome, Firefox, Opera, Edge and now in Safari too. In this article learn how to use WhatsApp Web with simple steps given below.

    WhatsApp Web Platforms: WhatsApp for iOS | WhatsApp for Android | WhatsApp for Windows PC | WhatsApp for Mac PC | WhatsApp for iPad | WhatsApp for BlackBerry | WhatsApp for Windows Phone | WhatsApp for Nokia | Symbian

    Android: Chats screen > Menu > WhatsApp Web.
    Nokia S60 and Windows Phone: Menu > WhatsApp Web.
    iPhone: Settings > WhatsApp Web.
    BlackBerry: Chats > Menu > WhatsApp Web.
    BlackBerry 10: Swipe down from the top of the screen > WhatsApp Web.
    Nokia S40: Swipe up from the bottom of the screen > WhatsApp Web.

    Know more about WhatsApp Web 



    WhatsApp Web using Android

    The First thing to do is to visit web.whatsapp.com and scan the QR Code that is shown.

    WhatsApp Web screen on PC

    On Android tap Menu(three dots) on the top right. From the list tap WhatsApp Web.

    WhatsApp using Android

    You will be taken to the next page of WhatsApp Web. You will see the list of computers you have logged in previously. Tap the ‘+’ icon on the top right to add a new device.

    WhatsApp using Android

    A QR Code scanner appears for you to scan the QR Code on the computer screen.

    WhatsApp using Android

    Once scanned you will see the WhatsApp mirrored to your PC screen with all the Chats and conversations.

    WhatsApp Web using iOS

    Similar to Android, You will first start with web.whatsapp.com from the browser on your computer.

    On your iPhone, go to WhatsApp App. On the bottom click settings > WhatsApp Web. You will go to the next page WhatsApp Web / Desktop.

    WhatsApp using iOS

    Here the list displays all your previous sessions in various computers. Click on one or start with a new one by tapping Scan QR Code from the bottom.

    WhatsApp using iOS

    Now QR Scanner opens for you to scan the QR Code from the browser.

    WhatsApp using iOS

    As soon as you do this, you will be taken to the WhatsApp on your PC.

    WhatsApp Web using Windows Phone

    On your PC visit web.whatsapp.com. The WhatsApp Wen page appears with a QR Code as shown in the picture above.

    Get your Windows Phone now and go to WhatsApp App.

    In the Chat screen, tap Menu in the bottom right. Select WhatsApp Web.

    WhatsApp using Windows Phone

    Now a screen appears with an image preview to scan the QR Code. Click ok, got it from the bottom left.

    WhatsApp using Windows Phone

    The QR Scanner opens, and you can scan the code from the PC screen.

    WhatsApp using Windows Phone

    This will straight away take you to the WhatsApp Web.

    WhatsApp Web Features

    WhatsApp Web brings the entire set of features from the WhatsApp mobile App. Thus there is nothing left in experiencing WhatsApp on PC.

    WhatsApp Features

    • As we are using WhatsApp Web from a PC it is obvious that the desktop notifications comes handy. To activate click Ok, Got It and Click Allow from the pop-up that appears as you open for the first time.
    • A chat icon to Create New Chat.
    • The Menu has New Group, Profile and Status, Archived, Setting and Logout.
    • Change your Profile and status by clicking your profile picture. You can also change your Name, profile picture by uploading a file from your PC and the status when needed.
    • Archive messages feature to save messages from WhatsApp Web.
    • Settings contain the Notifications, Chat Wallpaper, Blocked and Help.
    • Customise your chat screen with Chat Wallpaper.
    • Blocked shows your unwanted blocked contacts.
    • Help to assist you out with FAQ and other supports.

    Stay Safe With WhatsApp

    WhatsApp Web uses QR Codes for logging, which is a way better idea for signing in rather than using conventional mail id’s and passwords. The QR Code is a 2D code that can store a lot of information than the usual bar codes and is more secure too. Thus WhatsApp Web is very secure with this process. You can use your smartphone to scan the QR Code and get into the application while accessing out of your mobile.

    WhatsApp security

    As you are done with your WhatApp application, make sure to log out from the session without fail. If you have your computer protected with a password, then it os not going to be a problem but if you are in a situation to access your WhatsApp from any new computer then to ensure safety don’t forget to log out. To Logout tap Menu(three dots) > Logout. ou also have the option to log out from all the computers from the same page of login in.
