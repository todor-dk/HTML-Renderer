Free Language Translator is a very useful translating utility that is powered by Google Translate. The application enables you to translate texts between more than 40 different languages.

Free Language Translator has some useful features such as automatic source language detection and an accuracy calculator. It also has the option to create a personal dictionary with corrected translations.

Key features include:

  • Standard drag and drop explorer interface.
  • Over 40 languages supported.
  • Google Translate powered.

Overall, Free Language Translator is really easy to use, has a standard explorer interface with drag and drop support and some personalization options such as custom text font and the ability to pick another keyboard layout when translating to languages.
