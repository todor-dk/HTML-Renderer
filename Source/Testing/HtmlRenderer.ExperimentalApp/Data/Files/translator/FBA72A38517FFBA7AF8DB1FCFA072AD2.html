*THE MICROSOFT TRANSLATOR API IS NOW AVAILABLE ON THE AZURE PORTAL*

 

The Microsoft Translator Text API offer is listed in the Cognitive Services section of the Azure portal.

IMPORTANT: An Azure account is required. Read the steps to get started on the portal here.

If you subscribe on DataMarket today, minor auth code changes are required to move to the Azure portal.

New subscribers to the Azure portal can sign-up for an Azure account at Azure.com and receive a $200 credit to spend on Azure services for 30 days. This includes the Microsoft Translator API!

 

The Microsoft Translator Text Translation covers two types of API integration:

1) Text-to-Text translation: Enables translation of text between any of the 50+ supported languages.

2) Text-to-Speech synthesis: Outputs audio in any of the supported text-to-speech languages.

Additional functionality includes language detection and customization.

Example apps are in GitHub.

Add real-time end-to-end speech translation with the Microsoft Translator API Speech Translation.
