English Translation Instructions:
  1. Type in English or paste in English text from an electronic document. The free English translation can translate up to 800 characters per translation.

  2. Select appropriate input and output options by clicking on the "Show Options" button. Such as removing certain characters to clean up the input for translation. Select the Chinese character output – Simplified or Traditional characters and font size.

  3. Click the button "Translate English Now".

  4. When the translation appears, it is possible to have multiple English to Pinyin and English to Chinese translations per single English word. Choose the appropriate word by selecting its checkbox. This will hide other translations for that English word.

  5. To redisplay the possible translations, click on the [+] beside the word to re-expand.

  6. Click on the Print Translation Link provided when all sections have been completed to remove all hide/show links. This will provide a cleaned up version in a print window for printing/copying.
