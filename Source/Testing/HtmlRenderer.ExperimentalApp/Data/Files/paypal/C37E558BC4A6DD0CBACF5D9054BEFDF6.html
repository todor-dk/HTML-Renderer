Labels

Your shipping label contains all your necessary postage information and allows you to send your package. The label is affixed to your package once you have selected your service type, weight and options such as insurance and Signature Confirmation™ service.

Package List

A package list is a reference sheet that you can print after your labels have been printed. Each label contains a Package ID number that is listed on the package list. This reference allows you to quickly associate printed labels with packages to ship.

Packing Slips

A packing slip is a printable shipping receipt for the purchase of physical goods. It contains details of the items being shipped and can be placed in the package containing the purchase items.
