      Um Echtgeld spielen - PayPal

      Was ist PayPal?

      PayPal ist eine E-Wallet, die Ihnen sichere Einkäufe bei vielen Online-Anbietern gestattet. Sobald Sie Ihr PayPal-Konto eröffnet haben, können Sie Einzahlen, Abheben und Guthaben an viele Online-Anbieter überweisen, die Online-Zahlungen via PayPal akzeptieren - dazu zählt auch PokerStars. Unter www.paypal.com können Sie ein kostenloses Konto eröffnen, das an Ihre E-Mail-Adresse gekoppelt ist.

      PokerStars erlaubt nur ein PayPal-Konto pro PokerStars-Konto.

      Wie lade ich ein Guthaben auf mein PayPal-Konto?

      Es ist nicht erforderlich, vor einer Zahlung ein Guthaben auf Ihr PayPal-Konto zu laden. Verknüpfen Sie Ihr PayPal-Konto einfach mit Ihrer VISA-Karte, MasterCard oder Ihrem Bankkonto (nur Großbritannien) und zahlen Sie im PokerStars-Kassenbereich ein. Sie werden dort um Angabe Ihrer bevorzugten Zahlungsmethode gebeten. Wir akzeptieren keine American Express-Karten.

      Beachten Sie bitte, dass Sie zur Einzahlung auf PokerStars Ihr PayPal-Konto in Großbritannien oder Irland registrieren müssen. Sollten Sie nicht sicher sein, in welchem Land Sie Ihr PayPal-Konto angemeldet haben, wenden Sie sich bitte an den PayPal-Kundenservice unter 0870-7307-191 (aus Großbritannien) und 1890 943 220 oder 01 4369023 (aus Irland). Alternativ finden Sie hier ergänzende Kontaktinformationen.

      Wie mache ich mit PayPal eine Einzahlung bei PokerStars?

      Für eine Einzahlung gehen Sie bitte wie folgt vor:

      1. Gehen Sie in den PokerStars-Kassenbereich.
      2. Klicken Sie mit der gewünschten Währung auf "Chips kaufen".
      3. Wählen Sie "PayPal" und klicken Sie auf "Einzahlen".
      4. Geben Sie im anschließenden Fenster den Einzahlungsbetrag sowie Ihren Bonuscode (falls vorhanden) ein. Klick anschließend auf "Senden".
      5. Ihr Browser führt Sie nun zur PayPal Website. Folgen Sie zur Durchführung Ihrer Einzahlung den Anweisungen auf dem Bildschirm.
      6. Sobald Ihre Zahlung genehmigt wurde, schließen Sie das Browser-Fenster und kehren Sie zurück in den PokerStars-Kassenbereich. Klicken Sie zur Durchführung Ihrer Transaktion auf "Weiter".
      7. Ihr PokerStars-Kontostand wir nach Eingang der Zahlung aktualisiert.

        Welche Einzahlungslimits gelten für PayPal?

        Informationen zu den aktuellen Einzahlungslimits erhalten Sie im Kassenbereich.

        Kann ich auch Auszahlungen vornehmen?

        Bevor Sie eine Auszahlung auf Ihr PayPal-Konto veranlassen, vergewissern Sie sich bitte, dass ihm eine gültige Finanzeinrichtung zugewiesen ist (Bank oder Karte) und Ihre E-Mail-Adresse von PayPal verifiziert wurde.

        Sofern Sie innerhalb der letzten sechs Monate mindestens eine Einzahlung über PayPal durchgeführt haben, können Sie Ihr PokerStars-Guthaben auf Ihr PayPal-Konto auszahlen lassen.

        Bitte denken Sie daran, dass Auszahlungen vor einer Buchung auf Ihr PayPal-Konto zunächst automatisch - in nachstehender Reihenfolge - auf eine oder alle der folgenden Zahlungsmethoden zurückgebucht werden, die Sie möglicherweise für eine Einzahlung genutzt haben:

        1. CFT-fähige Visa- und Entropay-Karten
        2. NETELLER

        Ausführliche Informationen finden Sie in unseren Auszahlungsoptionen.

        Wie funktioniert die Wechselkursumrechnung?

        PokerStars orientiert sich bei der Umrechnung Ihrer Einzahlung in Ihre Kontowährung an den aktuellen XE-Durchschnittskursen. Für den Vorgang erhebt PokerStars gegebenenfalls eine geringe Gebühr. Auf der Seite Allgemeine Geschäftsbedingungen für Echtgeld-Transaktionen und Währungsumrechnungen finden Sie zusätzliche Informationen.

        Was kostet mich die Nutzung von PayPal bei PokerStars?

        PokerStars erhebt für die Nutzung von PayPal keinerlei Gebühren. Die Spieler tragen jedoch alle Kosten, die von PayPal berechnet werden.

        Hilfe bei Einzahlungen

         

        Die PokerStars Einzahlungs-Helpline 'Deposit Assistance Department' (DAD) hilft Ihnen bei Ihrer Echtgeldeinzahlung. Falls Ihre Einzahlung abgewiesen wird, haben Sie die Möglichkeit, einen Live Support anzufordern.

        Ein PokerStars-Mitarbeiter wird Sie via Live-Chat kontaktieren und Sie bei der Durchführung Ihrer Einzahlung unterstützen. Falls Sie einen Live Support wünschen und diese Option nicht angezeigt wird, wenden Sie sich bitte andad@pokerstars.eu und tragen Sie 'Chat Request' in die Betreffzeile Ihrer E-Mail ein. Für den Live Support müssen Sie eingeloggt bleiben.
