About Factory Automation

  • Description
  • Changes
  • Other Downloads
  • Screenshots

This Mod Adds Extractors To The Game Which Extract Items From The Block Above.  In The Future It Will Auto Eject.  Also It Will Have Different Tiers Of Extractors To Mine Faster.  Now You Can Change The Values In The Config
