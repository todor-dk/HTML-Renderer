FAS provides turnkey industrial robot solutions to America's leading manufacturers. We focus on machine tending, material removal, material handling and assembly applications. Our booth will showcase a machine tending cell that not only loads parts into a machine tool, but also automatically changes the chuck jaws for different size parts. In addition to machine tending functions, the demo will show the robot performing secondary operations for deburring and parts inspection.
