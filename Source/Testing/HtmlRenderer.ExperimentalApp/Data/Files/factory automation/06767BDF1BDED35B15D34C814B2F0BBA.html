Test Manager is our premier product. Test Manager enhances yield and throughput by reducing errors and test time. Test Manager standardizes data output for wafer-sorting and package testing for multiple test platforms in factories with diverse toolsets.  A variety of Wafer-Probe equipment, and Package-Handlers are supported, enabling any facility to standardize its data flows.

Integration Solutions for:

Wafer Probe & Final Test


FA Tools provides a focused portfolio of Test Cell Integration solutions and products for the semiconductor wafer probe and final test environment. We provide factory tool integration services and custom software applications to fit your specific factory needs.
Consulting & Custom Software Solutions

On-site factory custom support for a variety of FAB and Test equipment. GEM/SECS/HSMS custom programming for FAB or Test equipment. Custom GPIB drivers for machine control. Equipment performance reporting (EI)  & custom tool integration applications. Wafer-Sort or Package-Level Test Applications are our specialty.
