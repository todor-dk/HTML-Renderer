Factory Automation : Inverter | Cypress Semiconductor

Factory Automation : Inverter

  • Block Diagram
  • Design Considerations
  • SOFTWARE AND DRIVERS

Block Diagram

FM4 family S6E2Cx series MCU is ARM Cortex-M4 CPU incorporated embedded microcontroller. Unleash your system design due to CPU performance and large memory integration in addition to several sets of industry standard peripherals. The specific functionality of the Inverter has been a key application for industrial motor control. A major benefit of using an inverter for motor speed control is the energy savings. Inverters can accurately control motors over a wide speed range and keep the motor from stopping and restarting, which drains energy. The MCU is a key to control the inverter application.

Design Considerations

Inverter control system requires for MCU to execute multiple inverter motor control by integrated timer, PWM, ADC and Quadrature decoder for establishing real time feedback loop for precision motor control. The Inverter unit can be connected with Programmable Logic Controller (PLC) as a part of FA network.

Supplementary, 5V supply or I/O tolerant is still needed for several systems. Ambient temperature (Ta) requirement is higher than consumer applications such as -40C to +105C. High pin count GPIO is required for connecting to peripheral devices in addition to prefer QFP package rather than BGA package from solderability and reliability point of view.

S6E2Cx MCU can provide a true single chip solution because of integrated peripherals and CPU. ARM Cortex-M4 based 200MHz CPU power, 2MBytes high speed access on chip flash memory and 256Kbytes on chip SRAM. In addition, up to 3 units Multi Function Timer (MFT), 4 channel Quadrature Position/Revolution Counter (QPRC) and 32 channel high speed ADC can interface inverter motor unit. IEEE1588 compliant 10/100 base Ethernet MAC supports to connect PLC unit via variety of industrial standard protocols. Hardware cryptographic engine (AES-128/192/256, SHA-256, PKA) enables secure data communication with minimal CPU process overhead.

Development Kits/Boards
  1. Customer Starter Kit (SK-FM4-176L-S6E2CC-ETH):
    The FM4-176L-S6E2CC-ETH Starter Kit is a low-cost development platform for S6E2CC MCU. The board is equipped with peripheral devices that enables you can use to explore the modules on the MCU.
  2. Customer Development Kit (SK-FM4-216-ETHERNET):
    The FM4-216-ETHERNET is a full featured development platform for developing applications using FM4 S6E2CC MCU. This board provides a variety of peripherals for evaluating different modules of the MCU.

SOFTWARE AND DRIVERS

  • Flash Programmer

    The FLASH MCU Programmer is an application software for Cypress MCU which is able to program MCU on chip flash memory via PC. A programmed code can be read from MCU on chip memory to PC as well.

  • Pin and Code Wizard

    The "Pin and Code Wizard" is the tool helps to set the IO ports for Cypress 32-bit ARM Cortex microcontroller, and it has the following features. After activating the peripherals and GPIOs, which are used, the tool assigns the peripherals to the corresponding pins automatically. Customize easily the result of Pin assignment by graphical user interface. The tool outputs the source code according to the Setting of IO port for peripheral library function calls.

  • Sample Program (Peripheral Driver Library)

    The Sample Programs are available for quick MCU functionality verification and prototyping work. The Peripheral Driver Library (PDL) avoids you from writing MCU peripheral driver software from the scratch. It is also useful as a reference code when your code does not work correctly with some reason.
