Email Client Setup: Outlook

The setup process is a bit different for each version of Outlook, and many of them support Autodiscover, which automatically detects some or all of the required server settings from your username and password. Click on a link from the lists below for a step-by-step tutorial specific to the version of outlook you are using. If you aren't sure which version of Outlook you are using, see What version of Outlook do I have? for more information.

  • Outlook 2016 for Windows
  • Outlook 2016 for Mac
  • Outlook 2013
  • Outlook 2011
  • Outlook 2010
  • Outlook 2007
  • Outlook 2003
  • Outlook 2000
  • Outlook Express
  • Windows Live Mail
  • Windows 10 Mail
Using an email client not listed here? See Email Application Setup for help setting it up.
