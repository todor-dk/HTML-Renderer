Campus Email Web Access

Use the links below to access your mail from anywhere using a web browser.

Faculty and Staff

https://webmail.salisbury.edu/

Faculty and staff will use Outlook Web App to check their Outlook (Exchange) email online. Internet Explorer is recommended.

Attachment size limits: Microsoft Outlook has a 20 MB size limit for total mail message size (mail message + attachment) for sending and receiving. Max size for sending to pre-populated groups is 3MB.

Note: Some features and functionality, like the Address Book, are not available when using Google Chrome.

SU Students

www.outlook.com/gulls.salisbury.edu 
(log in with your full email address as your username, e.g. sstudent1@gulls.salisbury.edu)

If you need to change your password, visit our Password Assistance page.

Attachment size limits: Microsoft Outlook has a 20 MB size limit for total mail message size (mail message + attachment) for sending and receiving.

Note: SU student email accounts are deleted one year after your last semester at SU.
