Outlook: Set up email

Now that your Office 365 account is ready, let's set up your email in Outlook. Choose the version below that you want to use. You can find out which version of Outlook you have, if you don't know.

Outlook 2003 is not supported by Office 365. Please update your version of Outlook before proceeding.

Computer Operating System Outlook version
Windows 2007, 2010, 2013, 2016
Mac 2011, 2016

More info

  • If you have Windows, you can set up Outlook automatically, with the Outlook Setup Assistant.
  • If you use a Mac, you can instead set up your email using Apple Mail.
