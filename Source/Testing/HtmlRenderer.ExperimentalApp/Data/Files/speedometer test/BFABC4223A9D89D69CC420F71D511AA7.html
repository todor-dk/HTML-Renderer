Bandwidth Usage and Internet Connection Speed Test

Below is our list of sites for testing your internet connection speed and bandwidth usage.

Site Description Results example Additional notes

www.speedtest.net 		Broadband Internet connection analysis tool with large selection of testing servers that are geographically dispersed Works in any web browser that supports Flash

Speakeasy's Test 		Measures download and upload rates from several geographical locations Supports any web browser with Flash (at least version 7) plugin

Mcafee's Speedometer 		Simple Internet Connection Speedometer No Flash plugin required

internetfrog.com 		Java based download, upload, RTT test Java 2 Runtime Environment (J2RE) Plugin is required

performance.toast.net 		Simple Internet connection speed test Should work in any browser

dslreports.com Test 		Allows to choose internet connection testing method (Flash, J2RE, mobile test) Should work in any browser

ZDNet's Test 		Simple download test Needs browser with J2RE plugin

testmy.net 		Smart download and upload speeds test To run this test your browser should support Java 2 Runtime Environment

CNET speedtest 		Quick download speed test Should work in most of Internet browsers
