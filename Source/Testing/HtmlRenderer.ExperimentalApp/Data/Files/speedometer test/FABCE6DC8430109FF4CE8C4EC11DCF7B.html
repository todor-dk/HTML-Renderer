    In the mid 90's the cable speedo was done away with if the odometer is electronic then the speedo works from the sensor on the transmission
    i would suspect if the sensor is the fault then the odometer would no longer count the miles and the turn signals would fail to self cancel
    so if the odometer is working i would suspect that the speedo is faulty
