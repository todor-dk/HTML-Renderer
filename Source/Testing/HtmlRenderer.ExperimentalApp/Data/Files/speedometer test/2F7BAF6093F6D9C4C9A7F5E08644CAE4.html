Speed Test

  • Help Desk
  • Internet
  • Speed Test

The PenTeleData speed test is a diagnostic tool that is designed to help our Technical Support staff evaluate the performance of your Internet connection within the PenTeleData network. In order to run the PenTeleData speed test, you will need to have the latest version of Java installed. If you do not see the java applet below, please visit java.com and follow the on screen instructions.

Questions regarding connection speed?

  • High-speed cable customers
  • High-speed DSL


Note: In order to achieve better speed test results, please ensure that you take the speed test on an average of three times by using the "Refresh/Reload" button within your web browser or by using keyboard shortcut keys. Internet Explorer users, select the "F5" key along the top of your keyboard. Firefox users should press the "Ctrl" key and then, choose "R" on the keyboard (Ctrl+R).

If you feel that you are experiencing slow data transfers on your Internet connection you can contact Technical Support. When contacting Technical Support, please be sure to reference the "Test Result #XXX" which can be found just below your test result speeds.
