SMF Converter es un software que convierte datos de archivo MIDI estándar (SMF) disponibles en el mercado en un formato compatible con los instrumentos musicales CASIO y que, a continuación, transfiere los datos convertidos al instrumento. Una vez que se han transferido los datos, éstos se pueden utilizar junto con la función de lección para la reproducción de lecciones. Para obtener más información sobre cómo utilizar datos convertidos, consulte la documentación del usuario que se proporciona con cada instrumento.
Partes de una lección
La parte de la lección para la mano derecha corresponde al canal MIDI 4 y la parte de la lección para la mano izquierda corresponde al canal MIDI 3. Antes de convertir datos SMF, debe utilizar un software secuenciador disponible en el mercado para comprobar los ajustes de los canales MIDI para los datos originales y cambiarlos en caso de que sea necesario.
Transferencia de datos
Para transferir datos convertidos es necesario establecer una conexión MIDI entre el instrumento y el ordenador. Haga clic aquí para obtener información sobre las conexiones MIDI. Para las conexiones MIDI, utilice una interfaz USB-MIDI disponible en el mercado. Recuerde que algunos ordenadores no admiten el uso de una interfaz USB-MIDI por lo que, antes de comprar una, deberá comprobar el controlador de host para saber si es compatible.

Advertencia a los usuarios de ordenadores con Windows® 98SE
No instale la versión de SMF Converter desde el CD-ROM que se incluye con los modelos CASIO que se muestran a continuación. Si lo hace, es posible que no pueda iniciar el software desde el icono del escritorio de Windows o puede tener otro tipo de problemas. En su lugar, descargue la última versión de SMF Converter a continuación.
*Aplicable a los modelos: LK-200S/300TV, PX-310.
