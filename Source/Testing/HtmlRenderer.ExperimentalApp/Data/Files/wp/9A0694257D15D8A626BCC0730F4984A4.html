Welcome to WP NET

Managed WordPress Updates

WP NET keeps your site up-to-date and secure. We test and install all WordPress core software updates automatically. We also install security updates for WordPress plugins.

Multiple Daily Backups

Every night we take a full backup of all files and databases — with instant restore. We also run daily database-only backups and store them off-server for 30 days.

Scalable Performance

Our NZ cloud servers are fine-tuned and optimised to run WordPress — and nothing else. No email, no DNS — nothing that doesn’t make WordPress go faster.

Professional Support

Unlike other hosts, our support team are all experienced WordPress developers. Don’t put up with poor support — get friendly, responsive help and advice from WordPress Experts.

Serious Security

We take security very seriously. All servers have firewall protection with Fail2Ban and are monitored 24/7. Only secure FTPs is allowed and we regularly scan and audit all sites.

Automated Malware Scanning

We regularly scan and check all websites for known malware and other security vulnerabilities. We also monitor security networks and news sources so we’re always up to date with emerging threats.

SSD SAN Disk Storage

WP NET servers use high-performance SSD SAN disk storage systems for blazing fast speeds, scalability and redundancy. Add 1GB to any hosting plan for only $2 per month.

24/7 Monitoring

We employ industry standard, multi-layered monitoring tools to ensure timely and accurate systems status information. WP NET support are on the job within seconds should any site or server go down.
