Can't load the Facebook login page

Ask a question
July 2017


The most popular social networking site across the world, Facebook, has redefined communication over the internet in a whole new way. Sometimes the user may face problems when trying to access Facebook. While logging from the desktop the user may get a message like 'Internet Explorer cannot display the page'. This error can happen if the webpage is stored in the system files of Windows. In that case, Facebook.com needs to be deleted. After that, the reload of the login page in Facebook will happen seamlessly without any display of error.

Can't load the Facebook login page


  • Issue
  • Solution


Issue


I can't load the Facebook login page from my deskstop...
"Internet Explorer cannot display the webpage"

Solution

  • Check if facebook.com is in C:\WINDOWS\system32\drivers\etc\hosts.
  • If it is the case, simply delete it to be able to access your login page again.
