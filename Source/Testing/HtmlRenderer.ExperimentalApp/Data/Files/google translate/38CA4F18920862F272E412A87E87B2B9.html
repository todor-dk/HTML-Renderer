Free Google Desktop Translator

Google Translate Desktop is a free portable desktop translator based on google translate. It can quickly translate almost any text in 58 languages, and pronounce for 34 languages of them. It also enables users to translate website directly. It monitors Windows clipboard and automatically translate the content to target language. Also you can import text file and export the translations to a new *.txt file by Google Translate Desktop. In one word, Google Translate Desktop is a smart desktop translator for Windows users.

With free Google Translate Desktop, you can easily get the online translation without opening Google Translate. You are able to read foreign articles, communicate with foreign friends on facebook or twitter, learn foreign languages, view foreign websites and write emails in foreign languages easily. As a free portable desktop translator, Google Translate Desktop takes a little disk space. You don't need to install it to your PC like other free desktop translators. You can just run it and it will work for you directly.

Key Features

  • Pronunciation

    Google Translate Desktop provides pronunciation function to help with reading foreign language better. Built-in powerful pronunciation system supports both word and sentence pronunciation in a fluent voice.

  • Translate Webpage

    Google Translate Desktop can help you to translate a webpage for easily viewing. With this function, you can easily find the ree resources you want on a foreign website, no matter what language it is.

  • Clipboard Monitoring

    Google Translate Desktop can keep an eye on the clipboard. It will automatically detect the text on clipboard and translate it into target language without copy and paste the text to the translate box.

  • Automatically Identify the Input Text

    Google Translate Desktop automatically detects the input language. No matter what language the text really is, all the text will be translated into the target language as your demand automatically.

  • Support Translating Imported Files

    With Google Translate Desktop, you don’t need to open and copy the content from a TXT to the translate box. It supports translating the importing files directly, including *.txt, *.ini, *.log, *.inf file formats.

  • Support to Export Translation Results

    You don’t need to copy the translation results and save it by hand. Google Translate Desktop enables you to export the translate result to *.txt files. This will be very helpful in long article translating.

  • Support Romanization

    Some languages maybe pronounced by Romanization. Many other translate tools don't support Romanization, which is the advantage of Google Translate Desktop over its follower.

  • Support Online Dictionary

    As a smart desktop translating tool, Google Translate Desktop also supports online dictionary. Results are quoted from Google Translate online, even uncommon words can be found in Google Translate Desktop.

  • Live Update

    The updating process is realized only by one click. Once there is a new update, the program will notify you by a pop-up window. Press update on the pop-up window, and the program will update automatically.

  • Free for Use

    Google Translate Desktop is totally FREE for lifetime use. You don’t need to pay for a charged translator or dictionary, and the updating service is free too. If you like it, please help us recommend it to your friends.

  • Safe and Clean

    It is just a desktop program which you don't need to install or uninstall on your computer. Just put this program on your Windows Desktop and run it when translation is needed. Also it is nearly 0 CPU occupied.

  • Compatible with Windows 7

    Google Translate Desktop has abundant running experiences on Windows NT, 2000, XP, Vista and Win7. Highly-appreciated compatibility makes it more convenient and versatile for daily translation work.
