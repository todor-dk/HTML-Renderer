A reliable and user-friendly software solution that can be used whenever you want to quickly translate texts to other languages, with support for IE integration

Seeing the absolute immensity of the Internet, you are bound to run into a website that is written in an unknown language. When this happens, you can turn to ShaPlus Translator, as this software application helps you translate any desired text string to a huge list of languages (supported by the Google Translate service).

Simple setup process

The installation does not last very long as it does not offer you to download any third-party products, yet you should know that during this process you can choose to integrate ShaPlus Translator into Internet Explorer, as an add-on. Then you will be able to quickly translate any page content using the IE context menu.

In addition to that, you should know that you can also configure a keyboard shortcut to show and hide the program, a useful feature which lets you render ShaPlus Translator non-obtrusive with ease.

Change languages and view results

ShaPlus Translator can translate Clipboard content automatically and display the results in the main window.

Changing the translation languages is an easy task for any user, as all you have to do is to use the drop-down menus to choose the source and output languages. Moreover, you can set up the utility to automatically detect the source language, as well as set the chosen parameters as default.

Control font size, retranslate strings and copy them to the Clipboard

If you want to differentiate the source text from the translated one the application gives you the possibility to adjust the font size for each of them.

With ShaPlus Translator you can further translate parts or the entire text within the application itself. You just need to select the text you want to retranslate and right click the selection.

Last but not least, you should know that it is possible to copy the resulted body text to the Clipboard and thus use it with other tools.

Performance and conclusion

The computer’s performance is not going to be affected by ShaPlus Translator as it requires minimal resources. The response time is good and we did not come across any hangs or freezes in our tests.

All things considered, it is safe to say that ShaPlus Translator is a pretty simple, yet handy piece of software, easily accessible to all types of users, due to the intuitive interface it sports.
