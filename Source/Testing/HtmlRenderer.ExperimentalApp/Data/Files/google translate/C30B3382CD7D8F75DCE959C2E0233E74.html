Google™ Translator is a handy multi-language translator built on top of Google Translate. It gives you Google Translate UI feel right on your browser toolbar or as an injected iframe in pages. The extension provides translation, pronunciation and bookmarking all in one place. For bookmarking to work, make sure to be logged-in into your Google account.

Features

  • Easy one-click translation via Google Translate
  • Auto detects input language (no need to set input language each time)
  • Pop-up "bubble" translator is available on all web-pages
  • Translate selected text by holding Alt or Insert key
  • Context-menu web-page translation
  • Load requested features from GitHub

FAQ

  1. What is Google™ Translator extension and how can I use it?

    http://blog.add0n.com/2015/05/10/a-review-on-google-translator-extension.html

  2. What's new in this version?

    Please check the Logs section.

  3. How can I save a translation result for later use?

    When a translation is displayed on the panel or in the injected bubble, if you’d like to access the translation again in the future in the Google Translate website, touch the star icon.

  4. Why text-to-speech is not working on my Opera?

    To enable text-to-speech feature, you need to have a media player plug-in ( with mp3 support) installed on your Opera. Please follow this instruction to install QuickTime plug-in.

  5. Here are some tips for new users:

    Before using the extension, please make sure to visit the options page to adjust your settings

    To access the options page, click on the extension icon and select options.

    Note that all changes in the options page are auto saved.

    To activate the extension after installation, please reload all your existing pages.

  6. What is the alternative language option available in the Settings?

    When source language is set to "auto", if the detected language is equal to the destination language, alternative language will be used for translations.

  7. How can I fix "This developer's server appears to be busy. Please try again later." during Safari installation?

    The extension is hosted on GitHub. You can download it manually, https://github.com/andy-portmen/Simple-Translate/blob/master/src/dictionary.safariextz?raw=true.

    Please note that Safari extension will not be updated any more. Safari team has decided to charge developers around 110$ USD per year to obtain a developer certificate. This is against my mission to provide this product free-of-charge.

  8. Google™ Translator stopped playing sounds in the latest Firefox version. Is there any workaround for this?

    Most likely you have disabled the media.autoplay.enabled preference. For the extension to be able to play sounds, this preference needs to be enabled.

Editorial Review

The Google translator app is one of the most powerful built-in translators available for all major browsers. This extension works with Firefox, Opera, Safari and Chrome browsers to provide one click translations on any international webpage. While Google has a number of features in its search engine that make it simple to quickly translate webpages; the one click translation in the Google translate extension will automatically perform this action.

One reason that might interest you on getting Google translator app in your user interfaces is because it will auto detect input language and complete the translation to your native language on every webpage you visit. There's no need to consistently click a dialog box and you can actually set the application to automatically translate any webpage menu, content and more. The main features of Google translator are as follows:

  • It's simple to translate any selected text by holding the alt or insert key
  • The ability to translate context menus in the user interface of a website
  • Pop-up bubble translation on specific words throughout webpages
  • The ability to auto detect input language and perform translations
  • One click translation for any website that you visit based off of your set language

If you regularly visit international websites or even if you are attempting to learn a new language having the pop-up options as well as the automatic translation options can be extremely useful. Activating the extension is extremely simple as well and with the auto detection software built into Google translator you can usually produce some favorable results. Rather than having to consistently use the search engine to find the webpages you want this extension really speeds up the process. This is probably the best free translation extension available online currently and you can even set up multiple alternative languages beyond your automatic translation preference for several different users or for quick translations into multiple languages. With all of this ease-of-use Google translator app is highly recommended as a full featured extension for your favorite browser.
