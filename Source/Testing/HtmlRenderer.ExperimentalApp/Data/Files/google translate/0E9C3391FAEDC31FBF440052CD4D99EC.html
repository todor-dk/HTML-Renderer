Select and translate

You can select text on any page and immediately translate it using shortcuts, double-click or context menu.

A translation appears directly on the page in the neat tooltip, where you can learn synonyms, use text-to-speech and even open the translation in a new Google Translate tab.

Translate using shortcuts

You can create a lot of language pairs and assign them a shortcut. Then, you can simply select text on pages and press the shortcut to see a translation.

Translate with double-click

You can double-click on a word on any page to see a translation of it.

Translate from the context menu

You can also select text and then press the right button of the mouse to open the context menu, where Instant Translate has corresponding buttons to translate selection.

Unpin the tooltip

You can unpin the tooltip and drag it over the page. Thereby, you can have arbitrary amount of unpinned tooltips on the page with words you don’t want to forget.

Quick-accessible pop-up window

You can always make a translation by hand in the pop-up window that can be opened by an icon the top bar of the browser.

Additional features

As any other Instant Translate app, browser extensions translate between 100 languages, have dictionary and offline history, support text-to-speech.
