    About this Add-on

    Google™ Translator is a fast and handy translator built on top of the Google Translate for everyday use.

    Note 1: Google™ Translator does NOT belong or relate to the official Google Translate application or website at all. It is an unofficial Addon that is developed and maintained independently.

    It supports translation to all available languages and provides 5 core functionalities:

    (1) Easy translation through a menu-bar popup window.
    (2) Translation of a single word though double-click within any webpage.
    (3) Translation of lengthy texts by selecting text while holding "Alt" key.
    (4) Transferring selected text with Google Translator though right-click context menu item.
    (5) Spell check and automatic corrections for the input word.

    Note 2: to copy text within the toolbar-panel, just select any text and right-click. Text will be copied to clipboard automatically.

    Note 3: to change toolbar-panel size please see the Addon's options page.
