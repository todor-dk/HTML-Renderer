In the meantime, get the Simple Video converter available at:
http://sourceforge.net/projects/svideoconverter/?source=dlp

The converter will arrive as a .rar file, so use WinRAR (available at http://www.rarlab.com/download.htm, it will be the first or second blue link depending on which system you are using) to open it.

Open the file. Double click on the .jar file to start to converter.

A sound will play when the .ogv file is done downloading.

In the converter, click "Add Files". Go to the place where you downloaded the .ogv file. Select it then click "Open"

The .ogv file will be displayed in the input area. Below it you can select which video format you would like to convert it to. They are in order from least to most disk space used.

Below that you can select which folder you would like to have the converted folder appear in.

After you have done that, just click the "Start" button and wait!

When the status area says that the video process is complete, just go to the place you specified and click on the file to play the video.

Congratulations! You have downloaded a video from this site!
