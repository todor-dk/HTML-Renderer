How to integrate a Popup?

Let your users to convert video in a separate window. You can set your own logo and link too! 


<head>
    ...
    <script src="http://www.youtubemp3script.com/api/youtubemp3script-v1.js"></script>
</head>
First of all you need to include javascript file in your page header. This library was made as small as possible and it will not impact your page load time.
Optional: You can download script from the following location http://www.youtubemp3script.com/api/youtubemp3script-v1.js and store it on your own server.
How to use? 
<a href="http://www.yt-mp3.com/watch?v=YOUTUBE_ID"
   onClick="return YoutubeMp3Script(this)">
    Download MP3
</a>

When script is added, you can add as many youtube to mp3 download links as you want. You only need to edit following attribute:
YOUTUBE_ID - set youtube video id or encoded youtube url. 

<a href="http://www.yt-mp3.com/watch?v=7N8b3NZSJoY" onClick="return YoutubeMp3Script(this)">Download MP3</a>

Example with encoded youtube url: 

<a href="http://www.yt-mp3.com/watch?v=https%3A%2F%2Fwww.youtube.com%2Fwatch%3Fv%3D7N8b3NZSJoY"
   onClick="return YoutubeMp3Script(this)">
    Download MP3
</a>

Try it: Download MP3

Note - https%3A%2F%2Fwww.youtube.com%2Fwatch%3Fv%3D7N8b3NZSJoY is encoded youtube url of https://www.youtube.com/watch?v=7N8b3NZSJoY.
You can use this tool, to encode url online: http://meyerweb.com/eric/tools/dencoder/
Or use php urlencode function on serverside http://www.php.net/urlencode/
Can I integrate it simpler?

Yes! You can skip all the scripts and let users download with a simple link. 

<a href="http://www.yt-mp3.com/watch?v=YOUTUBE_ID">Download MP3</a>

YOUTUBE_ID - set youtube video id or encoded youtube url here

You can also open link in a new tab by adding target="_blank" attribute. 

<a href="http://www.yt-mp3.com/watch?v=7N8b3NZSJoY" target="_blank">Download MP3</a>

Try it: Download MP3

Can I use API?
