How does AudioTube work?

Audiotube is a free service for receiving the audio track from videos. After you enter a YouTube link, the file can be saved immediately, since YouTube provides the audio files in AAC format.

AAC files have a high quality and can be played with almost any MP3 player. You can convert the file to MP3 format optional.

We do not store any files for a convertion in the future. We do not use YouTube APIs. Private copying after the German Seeded allowed.
