  • wot_42735

    09-09-2013

    This is an incredibly useful site. I use it to download Youtube videos that I don't really care about watching, just listening to, so that I can listen to them on the go. It protects you legally by keeping track of copyrighted material and if you inadvertently request some, gives an error message something like "Google doesn't want us to convert this video" so it's safe to use. The quality of the mp3s is perfectly adequate for anyone except the most horrible audiophile, but they probably wouldn't be using YouTube in the first place.

    Best of all, Youtube-MP3 is free to use, is supported by non-intrusive advertising, and does not require its users to sign up or log in. More websites should be like this one.

    There is the slight caveat, however, that it doesn't discriminate between regular and adult content, and will freely allow minors to access adult audio. This is the case even if SafeSearch is enabled on YouTube's website, should they obtain the URL to adult content. As with everything on the web, young children should be closely supervised when using this tool.
