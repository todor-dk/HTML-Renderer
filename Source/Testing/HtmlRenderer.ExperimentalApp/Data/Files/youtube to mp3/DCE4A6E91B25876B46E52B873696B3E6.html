YouTube to MP3

Submitted by Maria on 29 February, 2012 - 11:21

Category: 

  • Entertainment

Description of App: 

YouTube to MP3 is for users who like to listen to music on YouTube but want to do it offline or on their favorite portable player. You have found soundtrack of new movie on YouTube and want to save it to your music library as MP3. Then MediaHuman YouTube to MP3 Converter for Mac is a best choice for you.This software is extremely simple to use and allows you to extract audio track from your favorite videos and music clips. Besides of it this software is completely free. 

Free or Paid: 

Free

Version Of OS X App Was Tested On: 

10.7

Accessibility Comments: 

At first glance i thought this app was going to be inaccessible. I guess it is somewhat, but it is usable. All i want to do is convert my youtube videos to mp3 if i find something interesting. See below for an explanation of how this app works.

Usability: 

There are some accessibility issues with this app, but it can still be used if you are willing to tolerate these issues and learn how to work around them.

Other Comments: 

When you first launch this app, there is a tool bar and an area that says unknown. There are also preferences but they don't seem to read. it is possible though to find something on youtube and have it converted to mp3. Step one, find a video you want to convert and copy the url. 2. command tab to youtube to mp3. 3. press vo m and navigate to the file menu. There is only one item here and it's paste url. 4. if you have growl set to speak, you wil hear that a file was added and the file name. You can paste in as many urls as you like using this method. 5. now navigate to the edit menu and choose start all. You should hear growl tell you that xxx has been started and another notification when it's completed. The app creates a folder called downloaded by media human in your music directory. So, in summary. it doesn't seem possible to edit or read preferences with voice over, but if you're happy with the default settings, this seems like a simple app to get your videos in to mp3.
