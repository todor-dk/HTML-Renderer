WhatsApp

WhatsApp uses Apple emoji images for emoji support on all platforms, including Android, Windows and the web. This differs from most apps which use the native emojis available for each platform.

The latest release of WhatsApp (2.17.63) added emojis from Apple iOS 10.2 in February 2017. See which emojis are new in WhatsApp.

Browse the Apple emojis available in WhatsApp

A few emojis appear differently in WhatsApp to other platforms. The red heart emoji has a beating animation, and appears larger than the other emojis when sent in a WhatsApp message:

Emojis Flags England, Scotland, and Wales are supported in WhatsApp, which are not yet supported by other major platforms. These use non-standard country codes; which differs from the upcoming Unicode subdivision flag support.

WhatsApp for Web, Android and Windows use custom artwork for the middle finger emoji. This was added in 2015, prior to Apple supporting this emoji.
