Can't connect to the new OneDrive. How to troubleshoot?

up vote 0 down vote favorite

Since Microsoft has announced its switch from Skydrive to OneDrive, I (along with my teammates here at work) have been been unable to connect to the website. Everyone can access OneDrive from home, but something at work is blocking it and IT is saying that they don't block anything. Like I mentioned before, everything was fine prior to yesterday.

We can all access it fine via the SkyDrive app in Windows 8.x or from the iOS app. But the website is inaccessible.

In Chrome, I get this error: ERR_CONNECTION_REFUSED. When I hook in Fiddler, I get ERR_TUNNEL_CONNECTION_FAILED in Chrome. In Fiddler, I can see that Chrome is trying to tunnel into onedrive.live.com:443, but the result is [Fiddler] The socket connection to onedrive.live.com failed. <br />ErrorCode: 10061. <br />No connection could be made because the target machine actively refused it 127.0.0.1:443. Basically stating that the socket connection to onedrive.live.com has failed.

Once in a while (and I have no idea if this is related), Fiddler comes up with the following message box:

How do I troubleshoot this?

windows-8 google-chrome onedrive
