DropBox, Google Drive, Box.net, OneDrive, Rackspace… Все Поддерживаются

Posted on March 17, 2014 by Dmitry Konnov in Uncategorized

Последняя версия для Windows 6.55 поддерживает следующие новые облачные сервисы:

DropBox, Google Drive, Box.net, OneDrive, Rackspace

Сервисы что мы поддерживаем уже на протяжении времени включают:
Amazon S3, Amazon Glacier, Microsoft Azure, Google Docs

В добавок, может быть использован любой облачный сервис который работает по протоколу WebDAV, FTP, или SSH/SFTP.

Google Drive это естественная заменена Google Docs. Google Drive будет видеть главным образом те же самые файлы, так что вы можете просто переключить существующий профиль с Google Docs на Google Drive. Однако, существует разница в отношении нативных документов Google Docs и конвертаций документов. У Google Drive нет возможности конвертации документов. Преимущество Google Drive в том что он может хранить метку времени файлов (Last Modified) выгруженных на сторону сервера при помощи Syncovery. Google Drive так же реализует использование Recycle Bin для удаленных файлов по выбору.

Т.к. Google Drive работает по другому с конвертируемыми документами, он может быть не в состоянии заменить Google Docs API в некоторых случаях. Оба выбора доступны при выборе протокола. Когда вы переключаете протоколы, проверьте вручную правильность работы ваших задач перед тем как перевести их в автоматический режим.
