Google Drive & Google Drive for Work

Google Drive is Google’s cloud storage & collaboration service that allows users to upload & sync files and later access them from a web browser or mobile device. In addition to collaborate with existing files, Google Drive allows users to create and edit existing files from any device.

Google provides a business edition of Google Drive as part of Google for Work suite known as Google Drive for Work.

Benefits of using Cloudsfer to migrate content from Google Drive & Google Drive for Work

  • Migrate all file types
  • Keep hierarchy structure
  • Maintain sharing settings (when migrating to Box and SharePoint Online)
  • Perform bulk users migration
  • Administrators can migrate from their Google Drive for Work sub-accounts
  • Apply filters by file size, created and modified date

Cloudsfer supports migrations from Google Drive to the following systems:
Amazon Cloud Drive, Google Drive, Box, Dropbox, OneDrive, OneDrive for Business, SharePoint Online, Copy, WebDAV, Amazon S3, Mega, FTP, Microsoft Azure Blob Storage, Flickr, Picasa

Benefits of using Cloudsfer to migrate content to Google Drive & Google Drive for Work

  • Migrate all file types
  • Keep hierarchy structure
  • Maintain sharing settings (when migrating from Box and SharePoint Online)
  • Administrators can migrate to their Google Drive for Work sub-accounts
  • Perform bulk users migration to Google Drive for Work
  • Automatically handle migration challenges according to Google Drive’s restrictions

Cloudsfer supports migrations to Google Drive from the following systems:
Amazon Cloud Drive, Google Drive, Box, Dropbox, OneDrive, OneDrive for Business, SharePoint Online, Copy, WebDAV, Amazon S3, Mega, FTP, Microsoft Azure Blob Storage, Flickr, Picasa, Facebook, Instagram, 500px
