Description

LIVESCORE
Live scores directly from the source, the 1st live score service on the Internet, powered by LiveScore.com since 1998.
This application comes directly from LiveScore Ltd., the leaders and inventors of real-time delivery of live sport data and owners of the no.1 ranked football / soccer web site globally.
Enjoy the excitement of live scores on your iPhone, iPod Touch & iPad same way that 50 million users are enjoying our existing online services every month, and become part of our network.

Since its launch, the LiveScore iPhone application has been downloaded 14 million times !
Consistently ranked no.1 in Sports category in +/- 25 countries.

FEATURES
- Live football / soccer scores from leagues, cup competitions and championships worldwide
- Live tennis, basketball & hockey scores worldwide
- Live league tables for football, hockey and basketball with highlighted promotion/relegation phases (NEW)
- Fixtures and Results for individual leagues/tournaments
- Possibility to choose a default sport, which is shown when the app is started
- Push notifications
- Commentaries in football and tennis
- Injury time
- Line-ups, Substitutions, Match statistics
- Live cricket scores worldwide (international & domestic)
- More than 1000 live football / soccer games followed live every week during the season
- Detailed match information
- Select your own refresh period
- User friendly
- Time & Date localized
- Retina display support (high resolution graphics)
- iPad support
- Very fast and responsive even on slow connection
- Automatically connects to a server nearest to your location for fast response
- Minimal data transfers
- Bandwidth friendly with our data compression
- Lightweight
- Free

FUTURE VERSIONS
Current version is Free and provides unlimited, unrestricted access to our football / soccer, tennis, basketball, hockey & cricket live scores.
In future versions current functionality is not going to be restricted however we may introduce additional advanced features where you would have an option to upgrade by purchasing an add-on.

REQUIREMENTS
iOS 8.0 (or higher) required.

What's New in Version 3.0.1

Version 3.0.1
-----------
- Bug fixes & stability improvements

Version 3.0
-----------
- New modern design.
- Home page now allows to quickly visualise fixtures for different days - showing 3 days in the past, the current day and 3 days in the future.
- Added favourite matches as a separate page for each sport.
- Added favourite leagues - selected leagues & competitions will be shown on top of each page (as long as the page shows some matches from the selected league or competition).
- Added a slide-out menu to change sports & settings.
- Added News section for soccer.
- Push notifications for cricket.

Version 2.2.6
-----------
- Fixed a network connection problem that some users were experiencing when loading data for selected tournaments in the tennis menu
- Fixed scrolling to top

Version 2.2.5
-----------
- Added league tables to soccer, hockey and basketball match details

Version 2.2.4
-----------
- Fixed timezone in line-ups last update time

Version 2.2.3
-----------
- Added last update time to line-ups and an information whether the line-ups are confirmed (not all leagues are covered)
- Bug fixes

Version 2.2.2
-----------
- Graphical line-ups for selected tournaments
- Improved Live Commentary section - added a short list of important commentaries and a long list of all commentaries

Version 2.2.1
-----------
- Promotion / relegation phases in League Tables
- Added league / tournament Fixtures and Results
- Improved navigation on paged screens - added previous & next page names
- Additional info in football / soccer Match Details (referee, venue, substitute players)
- Re-designed Settings, added a possibility to choose default sport, which is shown when the app is started

Version 2.2.0
-----------
- Push notifications
- Commentaries in football / soccer
- Match details (Match Info & Commentaries) in tennis
- Injury time
- Showing additional points awarded to (or deducted from) a team in league tables
- Added basic accessibility
- Brief tutorial

Version 2.1.0
-----------
- added match statistics
- added line-ups
- added substitutions
- improved memory consumption and UI rendering speed

Version 2.0.2
-----------
- multiple low memory warning fix
- other minor fixes and improvements

Version 2.0.1
-----------
- fixed bug with aggregate score which was not displayed correctly in case of double digits
- fixed timezone bug in by-date menu listing

Version 2.0
-----------
- league tables (live) for football / soccer and hockey (tap on league name)
- football & hockey match details now show assists for selected leagues (tap on goalscorer)
- aggregate scores in 2-leg knock out games
- detailed penalties shootout information for selected tournaments
- better details in tennis (who is serving, tie-break scores, game scores)
- menu listing by country / tournament and now also by date (swipe left/right)
- menu listing by country / tournament search function
