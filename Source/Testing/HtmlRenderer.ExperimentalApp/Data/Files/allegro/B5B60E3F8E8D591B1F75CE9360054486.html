The following page was printed from RemoteCentral.com:

Programming remotes with the "PRG" button:

  1. Press and hold the [PRG] button for three seconds, until the LED turns on.
  2. Press the component button you would like to program. Use CABLE or AUX for an additional TV or VCR; use AUX for a VD player or audio equipment.
  3. Enter the 3-digit code number, followed by [ENTER]. The red LED should flash then turn off. If it stays on, try a different code.
  4. To test the code, press [POWER]. Your device should come on.
  5. To clear all programmed codes and return the remote to factory defaults, enter "911" as the code number, followed by [ENTER].
Programming remotes without the "PRG" button:
  1. Press the component button you would like to program. If you are adding an additional TV or VCR use the CABLE button.
  2. Press and hold both [ENTER] and the [0] buttons for four seconds, until the LED lights.
  3. Enter the 3-digit code number followed by [ENTER]. All mode indicators should light briefly. If they do not light, try again.
  4. To clear all programmed codes and return the remote to factory defaults, enter "911" as the code number, followed by [ENTER].
