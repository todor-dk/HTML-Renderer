Overview

Integration with Allegro Group auction websites (pol. Integracja z Allegro).

Watch DrupalCamp Wrocław 2014 session video (polish) or check presentation slides (polish) to find more about that module.

This module contains following submodules:

Allegro API
Contains only API functions to communicate with Allegro WebAPI.
Allegro Category
Support for categorization of auction items.
Allegro Field
Support for fields related to auction items.
Allegro Item
Support for auction items as Drupal entities.

Installation

  1. Unpack and move directory allegro to your modules directory.
  2. Enable Allegro API module and Allegro Item submodule in the modules list of your site.
  3. Go to Allegro » Settings (admin/allegro/settings) and set up login credentials for Allegro WebAPI service.
  4. Go to Allegro » Settings » Categories (admin/allegro/settings/categories) and press button Update categories.
  5. Go to Allegro » Settings » Fields (admin/allegro/settings/fields) and press button Update fields.
  6. Go to Allegro » Add Allegro item (admin/allegro/item/add) and create auction item.
